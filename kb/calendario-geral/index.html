Recibos Automáticos de Venda

Envie recibos de venda automaticamente para seus clientes após a compra. O recibo é enviado como um arquivo PDF por email.

Como ativar recibos de venda automáticos

  1. Vá em Pagamentos > Configurações
  2. Clique na aba Recibos
  3. Ative a chave "Ativar recibos de venda automáticos para pagamentos"
  4. Clique em Salvar

Como personalizar seus recibos de venda

  1. Vá em Pagamentos > Configurações
  2. Clique na aba Recibos
  3. Título Personalizado: Edite o título do seu recibo.
  4. Números de Recibo: Defina um prefixo e número inicial para seus recibos.
  5. Modelo de Email: Personalize o modelo de email usado para enviar os recibos.

Entrega de Recibos

  • Os recibos são enviados como arquivos PDF anexados a um email.
  • O email é enviado diretamente para o endereço de email do contato.

Usando Modelos Personalizados

Você pode usar modelos personalizados para enviar recibos. Use os valores personalizados disponíveis no construtor de emails para criar recibos personalizados e com a identidade visual da sua empresa.

Como desativar recibos de venda automáticos

  1. Vá em Pagamentos > Configurações
  2. Clique na aba Recibos
  3. Desative a chave "Ativar recibos de venda automáticos para pagamentos"
  4. Clique em Salvar

Perguntas Frequentes

P: Posso enviar recibos de venda automaticamente apenas para certos tipos de pagamento? R: Não, ativar os recibos de venda aplica-se globalmente a todos os pagamentos feitos através de formulários de pedido, agendamentos de calendário e faturas. Se você quiser um tratamento seletivo de recibos, você precisará desativar os recibos automáticos e enviá-los manualmente para pagamentos específicos.

P: Os recibos de venda podem incluir campos personalizados ou mensagens personalizadas? R: Atualmente, os recibos de venda não suportam a adição de campos personalizados ou mensagens personalizadas além dos detalhes padrão da transação. No entanto, você pode personalizar o modelo de email associado ao recibo de venda para incluir uma mensagem geral ou elementos de marca.

P: Os recibos de venda são enviados imediatamente após o processamento de um pagamento? R: Sim, os recibos de venda são enviados automaticamente ao cliente assim que o pagamento é processado com sucesso. No entanto, podem ocorrer atrasos dependendo de fatores de entrega de email fora do controle da nossa plataforma, como o servidor de email do destinatário.

P: Como posso verificar se um recibo de venda foi entregue com sucesso a um cliente? R: Você pode verificar se um recibo de venda foi entregue com sucesso verificando a seção Relatórios de Email dentro da sua conta. Para acessá-la, vá em Marketing > Emails > Relatórios de Email. Uma vez lá, você pode usar filtros para pesquisar pelo endereço de email do destinatário ou pela linha de assunto do email do recibo. O painel de relatórios mostrará o status da entrega — como se o email foi entregue, aberto ou devolvido — ajudando você a confirmar se seu cliente recebeu o recibo.

P: Posso alterar um número de recibo para um número menor? R: Não. Os números de recibo são gerados automaticamente e devem ser sempre maiores do que o último usado. Você não pode editar, redefinir ou sobrescrever.

P: Se eu desativar os recibos automáticos, ainda terei um registro do pagamento? R: Sim. Desativar o email de saída não afeta sua contabilidade interna. O sistema ainda registrará cada pagamento bem-sucedido na aba Transações para seus registros.

P: Esta configuração se aplica a pagamentos únicos e recorrentes? R: Sim. Uma vez que esta chave é desativada, o sistema irá parar de enviar emails de recibo automatizados para todos os tipos de transação capturados através de seus gateways de pagamento conectados.

Incorporando um Calendário Externo

Este guia explica como obter o código de incorporação de um calendário específico para usar em sites externos.

Como fazer

  1. Vá em Calendários > Configurações de Calendário.
  2. Localize o calendário que você deseja incorporar.
  3. Clique no ícone de "três pontos" ao lado do calendário e selecione Compartilhar.
  4. Na janela que aparecer, selecione a aba Código de Incorporação.
  5. Clique em Copiar.
  6. Cole o código em um elemento de código personalizado no site desejado.

Observação: Se você deseja incorporar um Calendário do Full Funnel em um funil ou página de site dentro da nossa plataforma, use o elemento "Calendário" no construtor de páginas. Ele permite escolher o calendário que você deseja exibir e o insere automaticamente, sem precisar adicionar o código manualmente.

Como exibir mais de um calendário em uma página

Este artigo explica como exibir múltiplos calendários em uma única página, seja através de links ou incorporando-os em um funil.

Como fazer

Usando Links de Calendário:

  1. Abra o link de cada calendário em uma nova aba, clicando no ícone de link correspondente.
  2. Copie o ID de um dos calendários. Em seguida, vá para o outro calendário, adicione uma vírgula após o ID existente e cole o ID do outro calendário após a vírgula. Pressione "Enter" para recarregar a página.

Incorporando múltiplos calendários em uma página de funil:

  1. Vá em Configurações > Widget de Agendamento e copie o código de incorporação do primeiro calendário.
  2. Vá para a página do funil, adicione um elemento "JS/HTML Personalizado", clique em "Abrir Editor de Código" e cole o código de incorporação do calendário.
  3. Em uma nova aba, volte para Configurações > Widget de Agendamento, altere o nome do calendário no menu dropdown para o segundo calendário e copie o ID desse calendário.
  4. Volte para a aba da página do funil com o editor de código do elemento "JS/HTML Personalizado" aberto, adicione uma vírgula após o ID do calendário existente e cole o segundo ID do calendário. Clique em "Sim, salvar".
  5. Salve a página do seu funil e visualize.
