Gerenciar arquivos de contatos

Este artigo explica como adicionar e gerenciar arquivos para contatos na nossa plataforma, abordando tanto uploads diretos quanto campos personalizados para uma abordagem mais organizada do gerenciamento de documentos.

Por que adicionar arquivos aos contatos?

Adicionar arquivos aos contatos na nossa plataforma permite armazenar documentos importantes diretamente no perfil do contato, como contratos, faturas e notas. Esse recurso agiliza o gerenciamento de clientes, mantendo todos os arquivos essenciais organizados e facilmente acessíveis.

Nossa plataforma oferece dois métodos para fazer upload de arquivos para um contato:

  • Diretamente pelo perfil do contato.
  • Usando um campo personalizado para uploads de arquivos.

Considerações importantes antes de criar um campo personalizado

Ao configurar um campo personalizado de upload de arquivo, você tem várias opções importantes para configurar para melhor organização e funcionalidade. Essas opções determinam como o campo se comporta, onde ele aparece e que tipo de arquivos ele pode manipular.

| Campo | O que ele faz Especifica os tipos de arquivos que os usuários podem enviar por meio deste campo personalizado. | Formatos suportados: PDF, DOC/DOCX, JPG/JPEG, PNG, GIF, XLS/CSV. Isso garante que apenas arquivos compatíveis sejam aceitos. | | Restrição de arquivo | Define o tamanho máximo de arquivo permitido para uploads.
