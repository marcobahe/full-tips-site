Configuração do Mailgun passo a passo - Configuração de domínio GoDaddy

Este guia detalha como configurar o Mailgun com um domínio GoDaddy para envio de e-mails através da nossa plataforma.

Como fazer

  1. Crie uma conta no Mailgun:
    • Acesse o site do Mailgun.
    • Verifique sua caixa de entrada e confirme o endereço de e-mail.
  2. Adicione um novo domínio no Mailgun:
    • Faça login no Mailgun.
    • Clique em Sending (Envio) > Add New Domain (Adicionar Novo Domínio).
  3. Escolha o tipo de domínio:
    • Se você tem o domínio suaempresa.com, pode configurar o domínio principal ou um subdomínio com o Mailgun.
      • Domínio principal: Não deve ser usado com Gsuite ou outro provedor de e-mail.
      • Subdomínio: Digite algo como QUALQUERCOISA.suaempresa.com. Exemplos:
        • mg.suaempresa.com
        • resposta.suaempresa.com
        • suporte.suaempresa.com
    • Configure o domínio ou subdomínio sob US (Estados Unidos), não EU (União Europeia).
    • Clique em Add domain (Adicionar domínio).
    • Mantenha a tela seguinte aberta, pois você precisará adicionar registros DNS ao seu domínio.
  4. Acesse os registros DNS no GoDaddy:
    • Faça login no portfólio de domínios do GoDaddy.
    • Clique nos três pontos ao lado do seu domínio e selecione Edit DNS (Editar DNS). Pode ser necessário rolar para baixo para ver essa opção.
  5. Adicione os registros DNS:
    • Clique em Add (Adicionar) para adicionar um novo registro. Você precisará adicionar 5 registros no total.

Adicionar o 1º registro TXT

  1. Tipo: Selecione TXT no menu de opções.
  2. Host: NÃO INCLUA O DOMÍNIO RAIZ.
    • Se você estiver configurando:
      • mg.suaempresa.com, o nome do host será mg
      • resposta.suaempresa.com, o nome do host será resposta
      • Se estiver configurando o domínio principal, como suaempresa.com, o nome do host será @
  3. Valor TXT: O mesmo para todos.
    • Cole o seguinte registro: v=spf1 include:mailgun.org ~all
  4. Clique em Save (Salvar).

Adicionar o 2º registro TXT

  1. Clique em Add (Adicionar) para adicionar um novo registro.
  2. Tipo: Selecione TXT no menu de opções.
  3. Host: Copie tudo desde o início até a parte do subdomínio, SEM INCLUIR O DOMÍNIO RAIZ.
    • Exemplos:
      • Subdomínio: Copie mx._domainkey.helpdesk
      • Domínio principal: Copie mailo._domainkey
  4. Valor TXT: Volte ao Mailgun e copie o 2º registro TXT.
    • Cole o valor TXT copiado.
  5. Clique em Save (Salvar).

Adicionar o 1º registro MX

  1. Clique em Add (Adicionar) para adicionar um novo registro.
    • Se você usa Gsuite para receber e-mails no domínio principal, use um subdomínio para o Mailgun.
  2. Tipo: Selecione MX no menu de opções.
  3. Host:
    • Se você estiver configurando:
      • mg.suaempresa.com, o nome do host será mg
      • resposta.suaempresa.com, o nome do host será resposta
      • Se estiver configurando o domínio principal, como suaempresa.com, o nome do host será @
  4. Points to (Aponta para): O mesmo para todos.
    • Cole os seguintes dados: mxa.mailgun.org
  5. Priority (Prioridade): 10 (o mesmo para todos).
  6. Clique em Save (Salvar).

Adicionar o 2º registro MX

  1. Clique em Add (Adicionar) para adicionar um novo registro.
  2. Tipo: Selecione MX no menu de opções.
  3. Host:
    • Se você estiver configurando:
      • mg.suaempresa.com, o nome do host será mg
      • resposta.suaempresa.com, o nome do host será resposta
      • Se estiver configurando o domínio principal, como suaempresa.com, o nome do host será @
  4. Points to (Aponta para): O mesmo para todos.
    • Cole os seguintes dados: mxb.mailgun.org
  5. Priority (Prioridade): 10 (o mesmo para todos).
  6. Clique em Save (Salvar).

Adicionar o registro CNAME

  1. Clique em Add (Adicionar) para adicionar um novo registro.
  2. Tipo: Selecione CNAME no menu de opções.
  3. Host: Volte ao Mailgun para copiar o nome do host. Copie tudo desde o início até a parte do subdomínio, SEM copiar o domínio principal.
    • Se você estiver configurando:
      • mg.suaempresa.com, o nome do host será email.mg
      • resposta.suaempresa.com, o nome do host será email.resposta
      • Se estiver configurando o domínio principal, como suaempresa.com, o nome do host será email
  4. Points to (Aponta para): O mesmo para todos.
    • Cole os seguintes dados: mailgun.org

  5. Clique em Save (Salvar).

  6. Verifique as configurações DNS no Mailgun:

    • Após adicionar os 5 registros, volte ao Mailgun e clique em Verify DNS Settings (Verificar Configurações DNS).
    • Clique no mesmo botão novamente se alguns registros ainda não mostrarem a marca de verificação verde.
  7. Obtenha a chave API do Mailgun:
    • Após adicionar e verificar todos os registros DNS, obtenha a chave API do Mailgun.
  8. Adicione a chave API na nossa plataforma:
    • Cole a chave API do Mailgun nas configurações de e-mail da nossa plataforma em Configurações > Integrações.
  9. Envie um e-mail de teste:
    • Envie um e-mail de teste para verificar se tudo funciona corretamente.

Perguntas Frequentes

Configuração do Mailgun passo a passo com HostGator

Este guia detalha como configurar o Mailgun com um domínio HostGator para envio de e-mails através da nossa plataforma.

Como fazer

  1. Crie uma conta no Mailgun:
    • Acesse Mailgun.com.
  2. Verifique seu endereço de e-mail:
    • Verifique sua caixa de entrada e clique no link de verificação do Mailgun.
  3. Adicione um novo domínio no Mailgun:
    • Faça login no Mailgun.
    • Clique em Sending (Envio) > Add New Domain (Adicionar Novo Domínio).
  4. Escolha o tipo de domínio:
    • Se você possui o domínio suaempresa.com, você pode configurar o domínio principal ou um subdomínio com o Mailgun.
      • Domínio Principal:
        • Se você adicionar o domínio principal, ele não deve ser usado com Gsuite ou qualquer outro provedor de e-mail.
      • Subdomínio:
        • Para configurar um subdomínio, digite algo como QUALQUER_COISA.suaempresa.com.
        • Exemplos:
          • mg.suaempresa.com
          • respostas.suaempresa.com
          • suporte.suaempresa.com
    • Selecione a região US (Estados Unidos), não EU.
    • Clique em Add Domain (Adicionar Domínio).
    • Mantenha esta página aberta, pois você precisará dela para o próximo passo.
  5. Adicione os registros DNS no HostGator:
    • Faça login no cPanel do seu HostGator.
    • Na seção Domains (Domínios), clique em Zone Editor (Editor de Zona).
    • Localize seu domínio e clique em Manage (Gerenciar).
    • Clique em +Add Record (+Adicionar Registro) e selecione Add "TXT" Record (Adicionar Registro "TXT").
  6. Adicione o primeiro registro TXT:
    • Name (Nome): Depende do seu domínio/subdomínio. NÃO inclua o domínio raiz.
      • Se você estiver configurando:
        • mg.suaempresa.com, o nome do host será mg.
        • respostas.suaempresa.com, o nome do host será respostas.
        • suaempresa.com (domínio principal), o nome do host será @.
    • Record (Registro): O mesmo para todos.
      • Cole o seguinte: v=spf1 include:mailgun.org ~all
    • Clique em Add Record (Adicionar Registro).
  7. Adicione o segundo registro TXT:
    • Clique em +Add Record (+Adicionar Registro) e selecione Add "TXT" Record (Adicionar Registro "TXT") novamente.
    • Name (Nome): Copie tudo desde o início até a parte do subdomínio. NÃO inclua o domínio raiz.
      • Exemplos:
        • Subdomínio: Copie mx._domainkey.helpdesk (apenas o que está em itálico).
        • Domínio principal: Copie mailo._domainkey (apenas o que está em itálico).
    • Record (Registro):
      • Volte para o Mailgun e copie o segundo registro TXT fornecido.
      • Cole o registro TXT longo copiado aqui.
    • Clique em Add Record (Adicionar Registro).
  8. Adicione o primeiro registro MX:
    • Clique em +Add Record (+Adicionar Registro) e selecione Add "MX" Record (Adicionar Registro "MX").
      • Se você usa Gsuite para receber e-mails no domínio principal, use um subdomínio para o Mailgun.
    • Name (Nome): Depende do seu domínio/subdomínio.
      • Se você estiver configurando:
        • mg.suaempresa.com, o nome será mg.
        • respostas.suaempresa.com, o nome será respostas.
        • suaempresa.com (domínio principal), o nome será @.
    • Priority (Prioridade): 10 (o mesmo para todos).
    • Destination (Destino): O mesmo para todos.
      • Cole o seguinte: mxa.mailgun.org
    • Clique em Save Record (Salvar Registro).
  9. Adicione o segundo registro MX:
    • Clique em +Add Record (+Adicionar Registro) e selecione Add "MX" Record (Adicionar Registro "MX").
    • Desta vez, adicione outro registro para mxb.mailgun.org.
    • Name (Nome): Depende do seu domínio/subdomínio.
      • Se você estiver configurando:
        • mg.suaempresa.com, o nome será mg.
        • respostas.suaempresa.com, o nome será respostas.
        • suaempresa.com (domínio principal), o nome será @.
    • Priority (Prioridade): 10 (o mesmo para todos).
    • Destination (Destino): O mesmo para todos.
      • Cole o seguinte: mxb.mailgun.org
    • Clique em Save Record (Salvar Registro).
  10. Adicione o registro CNAME:
    • Clique em +Add Record (+Adicionar Registro) e selecione Add "CNAME" Record (Adicionar Registro "CNAME").
    • Name (Nome): Copie do Mailgun. Copie tudo desde o início até a parte do subdomínio. NÃO copie o domínio principal.
      • Se você estiver configurando:
        • mg.suaempresa.com, o nome será email.mg.
        • respostas.suaempresa.com, o nome será email.respostas.
        • suaempresa.com (domínio principal), o nome será email.
    • Record (Registro): O mesmo para todos.
      • Cole o seguinte: mailgun.org
    • Clique em Save Record (Salvar Registro).
  11. Verifique as configurações DNS no Mailgun:
    • Volte para o Mailgun e clique em Verify DNS Settings (Verificar Configurações DNS).
    • Clique no mesmo botão novamente se alguns registros ainda não mostrarem a marca de verificação verde.
  12. Obtenha a chave API do Mailgun:
    • Depois de adicionar todos os registros DNS e verificar, obtenha a chave API do Mailgun.
  13. Insira a chave API na nossa plataforma:
    • Cole a chave API do Mailgun nas configurações de e-mail da nossa plataforma em Configurações > Integrações.
  14. Envie um e-mail de teste:
    • Envie um e-mail de teste para verificar se tudo funciona corretamente.

Requisitos

  • Acesso à conta Mailgun.
  • Acesso ao cPanel do seu domínio no HostGator.

Configuração do Mailgun Passo a Passo com Domínio Namecheap

Este guia detalha como configurar o Mailgun com um domínio Namecheap para envio de e-mails através da nossa plataforma.

Como fazer

  1. Crie uma conta no Mailgun:
    • Acesse Mailgun.com e siga o processo de inscrição.
  2. Verifique seu endereço de e-mail:
    • Verifique a caixa de entrada do e-mail utilizado no cadastro e confirme o endereço.

    • Faça login no Mailgun.
    • Clique em "Sending" (Envio) > "Add New Domain" (Adicionar Novo Domínio).
  4. Escolha o tipo de domínio:
    • Se o seu domínio for suaempresa.com, você pode configurar o domínio principal ou um subdomínio com o Mailgun.
      • Domínio Principal:
        • Se você adicionar o domínio principal, ele não deve ser usado com Gsuite ou qualquer outro provedor de e-mail.
      • Subdomínio:
        • Para configurar um subdomínio com o Mailgun, digite QUALQUER_COISA.suaempresa.com.
        • Exemplos:
          • mg.suaempresa.com
          • respostas.suaempresa.com
          • suporte.suaempresa.com

    • Clique em "Add Domain" (Adicionar Domínio).
  5. Adicione os registros DNS no Namecheap:
    • Faça login em sua conta Namecheap.

        @ 3600 10 ALT4.ASPMX.L.GOOGLE.COM

    • Vá para Site Tools (Ferramentas do Site) > Domain (Domínio) > DNS Zone Editor (Editor de Zona DNS).
    • Na seção Create New Record (Criar Novo Registro), adicione os seguintes registros:

    • Exemplo para domínio principal: copie mailo._domainkey
  4. Value (Valor): Copie o valor do 2º registro TXT fornecido pelo Mailgun.
  5. Clique em Create (Criar).

  1. Clique na aba MX > Selecione Add your own MX records (Adicionar seus próprios registros MX).

    • Para subdomínios (ex: mg.suaempresa.com.br), insira apenas a parte do subdomínio (ex: mg).
    • Para o domínio principal (ex: suaempresa.com.br), insira @.
  3. Priority (Prioridade): 10 (o mesmo para todos).
  4. Destination (Destino): mxa.mailgun.org (o mesmo para todos).
  5. Clique em Create (Criar).

Adicionar o 2º registro MX

  1. Adicione outro registro MX.
  2. Name (Nome): Depende do seu domínio.
    • Para subdomínios (ex: mg.suaempresa.com.br), insira apenas a parte do subdomínio (ex: mg).
    • Para o domínio principal (ex: suaempresa.com.br), insira @.

  4. Destination (Destino): mxb.mailgun.org (o mesmo para todos).
  5. Clique em Create (Criar).

Adicionar o registro CNAME

  2. Name (Nome): Copie a parte inicial do nome fornecido pelo Mailgun, até a parte do subdomínio, SEM incluir o domínio principal.

    • Para o domínio principal (ex: suaempresa.com.br), insira email.
  3. Resolves to (Resolve para): mailgun.org (o mesmo para todos).
  4. Clique em Create (Criar).
  5. Verifique as configurações DNS no Mailgun:
    • Volte para a página do Mailgun e clique em Verify DNS Settings (Verificar Configurações DNS).
    • Clique novamente se alguns registros ainda não mostrarem o status de verificado (check verde).
  6. Copie a chave API do Mailgun:
    • Após verificar todos os registros DNS, copie a chave API do Mailgun.
  7. Insira a chave API na plataforma:
    • Cole a chave API do Mailgun nas configurações de email da nossa plataforma.
  8. Envie um email de teste:
    • Envie um email de teste para verificar se a configuração está funcionando corretamente.
