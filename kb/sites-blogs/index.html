Configuração do Mailgun - Domínio no Siteground

Como fazer

Siga este passo a passo detalhado para configurar seu domínio do Siteground no Mailgun e integrá-lo ao Full Funnel.

1. Criar Conta e Adicionar Domínio no Mailgun: * Acesse Mailgun.com e crie uma conta. * Verifique seu endereço de e-mail através da caixa de entrada. * Faça login no Mailgun, vá em Sending > Add New Domain. * Escolha entre configurar o domínio principal (ex: empresa.com) ou um subdomínio (ex: mg.empresa.com). * Domínio Principal: Não deve estar em uso com Gsuite ou outro provedor de e-mail. * Subdomínio: Pode ser qualquer coisa, como mg.empresa.com, respostas.empresa.com. * Importante: Selecione a região US (não EU) ao configurar. * Clique em Add Domain. Mantenha a tela com os registros DNS abertos para o próximo passo.

2. Adicionar Registros DNS no Siteground: Acesse o Site Tools da sua hospedagem Siteground e siga para Domain > DNS Zone Editor. Você precisará adicionar 5 registros.

  • Para adicionar o 1º Registro TXT:

    1. Na seção Create New Record, clique na aba TXT.
    2. Name (Host): Varia conforme seu domínio. NÃO INCLUA O DOMÍNIO RAÍZ.
      • Para subdomínio mg.empresa.com: digite mg.
      • Para domínio principal empresa.com: deixe em branco ou use @.
    3. Value (Conteúdo): Cole este valor: v=spf1 include:mailgun.org ~all
    4. Clique em Create.

  • Para adicionar o 2º Registro TXT:

    1. Clique em + Add Record novamente e selecione a aba TXT.
    2. Name (Host): Copie APENAS a parte do hostname fornecida pelo Mailgun, sem o domínio raiz.
      • Exemplo do Mailgun: Se for mx._domainkey.helpdesk.empresa.com, copie mx._domainkey.helpdesk.
    3. Value (Conteúdo): Copie e cole o valor exclusivo do segundo registro TXT fornecido pelo Mailgun.
    4. Clique em Create.

  • Para adicionar o 1º Registro MX:

    1. Clique na aba MX e selecione Add your own MX records.
    2. Name (Host): Use o mesmo hostname do primeiro registro TXT (mg, @ ou em branco).
    3. Priority (Prioridade): 10
    4. Destination (Destino): Cole: mx.a.mailgun.org
    5. Clique em Create.

  • Para adicionar o 2º Registro MX:

    1. Adicione outro registro MX.
    2. Name (Host): Repita o mesmo hostname.
    3. Priority (Prioridade): 10
    4. Destination (Destino): Cole: mx.b.mailgun.org
    5. Clique em Create.

  • Para adicionar o Registro CNAME:

    1. Clique na aba CNAME.
    2. Name (Host): Copie APENAS a parte do hostname fornecida pelo Mailgun para o CNAME.
      • Exemplo do Mailgun: Se for email.mg.empresa.com, copie email.mg.
    3. Resolves to (Aponta para): Cole: mailgun.org
    4. Clique em Create.

3. Verificar e Finalizar: * Volte à tela do Mailgun e clique em Verify DNS Settings. * Se alguns registros não aparecerem com a marcação verde, clique novamente para verificar após alguns minutos. * Após a verificação bem-sucedida, copie a API Key do Mailgun (em Sending > Domains > [Seu Domínio] > API Key). * Adicione essa chave API nas Configurações do Full Funnel, na seção de Integrações de e-mail.

Requisitos

  • Uma conta ativa no Mailgun.
  • Acesso administrativo ao painel de controle do domínio no Siteground (Site Tools).
  • Se usar Gsuite no domínio principal, é obrigatório configurar um subdomínio para o Mailgun.

FAQ

Posso usar o mesmo domínio principal para o Mailgun e para meu e-mail corporativo (Gsuite/Outlook)? Não é recomendado. Para evitar conflitos, configure um subdomínio (ex: envios.empresa.com) para o Mailgun se seu e-mail principal já usa o domínio raiz.

Quanto tempo leva para os registros DNS serem verificados? Geralmente leva alguns minutos, mas pode demorar até 48 horas para propagar totalmente. Clique no botão de verificação no Mailgun periodicamente.

O que fazer se a verificação DNS falhar? Verifique se você copiou os valores exatos fornecidos pelo Mailgun, especialmente nos registros TXT e CNAME. Confirme se o "Name (Host)" está correto, sem incluir o domínio raiz (.empresa.com). Aguarde mais tempo para propagação.
