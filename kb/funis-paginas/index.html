Links e Botões de Clique para Ligar, Clique para SMS e Clique para Email

Este artigo mostra como adicionar links e botões de clique para ligar, clique para SMS e clique para email nos seus Sites e Funis do Full Funnel. Esses links clicáveis melhoram a conversão e facilitam o contato dos leads com um único toque, especialmente em dispositivos móveis.

Como fazer

Como Adicionar Links de Clique para Ligar

Para criar um link que as pessoas possam clicar/toque para iniciar uma ligação telefônica: 1. No editor de Funnel ou Site, destaque o texto que será o link. 2. Clique no ícone de link (cadeado). 3. No campo de link, insira o formato: tel:+5511999999999 (substitua pelo número com código do país e área). 4. Clique em Salvar. 5. Visualize e teste o link em um dispositivo móvel.

Como Adicionar Links de Clique para Email

Para criar um link que as pessoas possam clicar/toque para iniciar um email: 1. No editor, destaque o texto que será o link. 2. Clique no ícone de link (cadeado). 3. No campo de link, insira o formato: mailto:seuemail@exemplo.com. 4. Clique em Salvar. 5. Visualize e teste o link.

Como Adicionar Botões de Clique para Ligar

Para criar um botão que inicia uma ligação telefônica: 1. No editor, clique no botão que deseja configurar. 2. No painel lateral direito, localize a seção Ações do Botão. 3. No menu suspenso Vincular a, selecione a opção Ligar. 4. Insira o número de telefone no campo fornecido. 5. As alterações são salvas automaticamente. Visualize e teste o botão.

Como Adicionar Botões de Clique para SMS

Para criar um botão que inicia uma mensagem de texto (SMS): 1. No editor, clique no botão que deseja configurar. 2. No painel lateral direito, localize a seção Ações do Botão. 3. No menu suspenso Vincular a, selecione a opção SMS. 4. Insira o número de telefone no campo fornecido. 5. As alterações são salvas automaticamente. Visualize e teste o botão.

Como Adicionar Botões de Clique para Email

Para criar um botão que inicia um email: 1. No editor, clique no botão que deseja configurar. 2. No painel lateral direito, localize a seção Ações do Botão. 3. No menu suspenso Vincular a, selecione a opção Email. 4. Insira o endereço de email no campo fornecido. 5. As alterações são salvas automaticamente. Visualize e teste o botão.

Requisitos

  • Um Funnel ou Site criado no construtor do Full Funnel.
  • O número de telefone ou endereço de email que você deseja que seja usado no link/botão.

FAQ

P: Como posso rastrear cliques nesses links? R: Você pode usar Links de Gatilho para rastrear e automatizar ações com base nos cliques através de Automações.

P: O que acontece se o dispositivo do usuário não suportar o link de clique (ex.: link sms: em um desktop)? R: Se um dispositivo não suportar o protocolo, nada acontecerá ao clicar. É recomendado testar seus links em diferentes dispositivos e considerar incluir informações de contato alternativas.

P: Posso adicionar múltiplas ações em um único link (como ligar e enviar SMS)? R: Não. Cada link ou botão pode executar apenas uma ação por vez (ligação, SMS ou email). Para oferecer múltiplas opções, use links ou botões separados para cada ação.

P: Posso usar links de clique em imagens ou ícones? R: Sim. No Construtor de Funnel ou Site, você pode adicionar um hiperlink a imagens ou ícones usando os mesmos formatos (tel:, mailto:). Isso é útil para CTAs visuais, como ícones de telefone ou mensagem.

Página de Redirecionamento Instantâneo - Solução de Problemas de Funil

Como fazer

Para configurar uma página de redirecionamento instantâneo dentro de um funil, siga estas etapas:

  1. Acesse a seção Sites no menu principal.
  2. Selecione o funil onde deseja adicionar o redirecionamento.
  3. No editor de páginas, adicione um novo elemento ou página.
  4. Procure pela opção de Redirecionamento ou Página de Redirecionamento.
  5. Insira a URL completa (incluindo https://) para a qual os visitantes devem ser direcionados.
  6. Salve as alterações e publique o funil.

Requisitos

  • Você deve ter um funil criado dentro da plataforma.
  • É necessário ter permissões de edição para o funil em questão.
  • A URL de destino para o redirecionamento deve ser válida e acessível.

FAQ

P: O redirecionamento não está funcionando. O que fazer? R: Verifique se a URL de destino foi inserida corretamente, incluindo o protocolo (http:// ou https://). Certifique-se também de que a página com o redirecionamento foi publicada após a configuração.

P: Posso usar um redirecionamento para um domínio externo? R: Sim, a página de redirecionamento instantâneo pode apontar para qualquer URL válida na internet, seja dentro da sua conta do Full Funnel ou em um domínio externo.

P: O redirecionamento é permanente (301) ou temporário (302)? R: A funcionalidade padrão dentro dos funis geralmente configura um redirecionamento do tipo 302 (temporário). Para redirecionamentos permanentes (301), consulte o guia específico na base de conhecimento.

Menus de Navegação no Construtor de Funis

Como fazer

Para adicionar e configurar um menu de navegação no seu funil ou site, siga estes passos:

  1. Abra o seu Funnel ou Website no Construtor.
  2. Clique no botão "Adicionar Elemento".
  3. Selecione a opção "Menu de Navegação" na lista.
  4. Arraste o elemento para a seção desejada da sua página (geralmente no cabeçalho ou rodapé).
  5. Clique no elemento do menu para abrir suas configurações.

Para configurar os itens do menu: 1. No painel de configurações do menu, você pode adicionar novos itens. 2. Clique em um item para renomeá-lo e definir seu link (para uma página interna, URL externa ou âncora na página). 3. Para criar um menu suspenso (dropdown), adicione um item principal e, em seguida, clique em "Adicionar Subitem" abaixo dele. 4. Configure o link do subitem. Repita para adicionar mais níveis, se necessário. 5. Use os controles de arrastar e soltar para reordenar os itens.

Para ativar a opção de Quebra (Wrap): 1. Com o elemento do menu selecionado, vá até a aba "Configurar" no painel lateral. 2. Localize e ative a opção "Quebrar" (Wrap). Isso fará com que os itens do menu que não couberem na largura disponível sejam movidos para uma nova linha automaticamente.

Para personalizar o estilo: 1. Com o menu selecionado, vá até a aba "Estilo" no painel lateral. 2. Aqui você pode personalizar a fonte, tamanho, alinhamento, espaçamento, cores de fundo, efeitos de hover e a aparência dos menus suspensos.

Requisitos

  • Você precisa ter um funil ou site aberto no Construtor do Full Funnel.
  • O elemento "Menu de Navegação" está disponível na biblioteca de elementos padrão.

FAQ

P: O que faz a opção "Quebrar" (Wrap)? R: A opção "Quebrar" permite que os itens do menu de navegação passem para uma nova linha automaticamente se não houver espaço horizontal suficiente. Isso evita que o layout quebre e melhora a responsividade em telas menores.

P: As melhorias nos submenus exigem alguma alteração de configuração? R: Não. As melhorias no comportamento e estilo dos menus suspensos são aplicadas automaticamente a todos os menus.

P: A rolagem suave para elementos âncora está ativa por padrão? R: Sim. Quando você configura um item do menu para rolar até uma seção da página, a transição agora é suave por padrão.

P: Essas atualizações afetam funis ou sites já publicados? R: As melhorias de estilo e a rolagem suave beneficiam os menus existentes automaticamente. A opção "Quebrar" (Wrap) precisa ser ativada manualmente no construtor, se desejada.

P: Essas mudanças alteram minhas configurações de estilo personalizadas? R: Não. Todas as personalizações de estilo que você já aplicou aos seus menus permanecem intactas.

Adicionar Texto Dinâmico às Páginas de Funil

Como fazer

Para personalizar páginas de funil com informações específicas do contato, como o primeiro nome, siga estes passos:

  1. Acesse o Editor de Funil:

    • Na plataforma Full Funnel, vá até Sites e selecione o funil que deseja editar.
    • Clique em "Editar" para abrir o construtor de funis.

  2. Selecione a Página de Destino:

    • No menu de páginas do funil, escolha a página onde o texto dinâmico será exibido (por exemplo, uma página de agradecimento ou confirmação).

  3. Adicione ou Edite um Elemento de Texto:

    • Na página, clique para adicionar um novo elemento "Texto" ou selecione um bloco de texto já existente.
    • No painel de edição do elemento, clique na área de texto onde a personalização deve aparecer.

  4. Insira a Variável Dinâmica:

    • Com o cursor na posição desejada, procure pela opção "Inserir Dinâmico" ou "Variáveis" (geralmente representada por um ícone como {...} ou um sinal de + no editor de texto).
    • Selecione a variável correspondente ao campo do contato que você quer mostrar, como {{contact.first_name}} para o primeiro nome.
    • A variável será inserida no texto. Você pode combiná-la com texto estático, por exemplo: "Obrigado, {{contact.first_name}}! Seu download está pronto."

  5. Salve e Publique:

    • Após configurar, clique em "Salvar" no elemento e depois em "Publicar" no funil para que as alterações entrem em vigor.

Requisitos

  • Um funil criado na plataforma Full Funnel.
  • A página de destino (ex.: página de agradecimento) deve ser acessada após uma ação onde as informações do contato são capturadas (como um formulário de inscrição).
  • O contato deve ter o campo correspondente (ex.: "Nome") preenchido em seu perfil para que o texto dinâmico seja renderizado corretamente.

FAQ

O texto dinâmico não está aparecendo para o visitante. O que fazer? Verifique se o contato possui um valor registrado no campo usado na variável (como o primeiro nome). Se o campo estiver vazio, nada será exibido no lugar da variável.

Posso usar outros campos além do primeiro nome? Sim. A plataforma oferece variáveis para diversos campos padrão e personalizados do contato, como sobrenome, e-mail, telefone e campos criados por você.

Isso funciona em qualquer tipo de página do funil? Funciona em qualquer página dentro do construtor de funis. É mais comum e útil em páginas de confirmação ou agradecimento, onde a personalização tem maior impacto após a captura dos dados.

Importar ou Clonar Funis e Sites

Existem duas maneiras principais de mover ou replicar funis e sites dentro da plataforma Full Funnel: importando através de um link de compartilhamento ou clonando diretamente.

Como fazer

Método 1: Importar Funis ou Sites via Link de Compartilhamento

Este método é usado quando você recebe um link de um modelo de funil ou site de um criador externo ou marketplace.

  1. Acesso: Certifique-se de estar logado na sua agência no Full Funnel antes de clicar no link de compartilhamento.
  2. Abrir Link: Abra uma nova guia no navegador e clique ou cole o link de compartilhamento recebido.
  3. Selecionar Local: Na janela que se abre, selecione os subcontas (Locations) para as quais deseja importar o template. É possível selecionar até 3 locais de uma vez.
  4. Finalizar: Confirme a importação. O processo pode levar alguns minutos.

Aviso Importante: * Dê alguns segundos após a confirmação antes de atualizar a página. * Elementos de captura de leads, como formulários, pesquisas ou pop-ups, NÃO serão transferidos no processo de importação. Eles precisarão ser configurados manualmente após a importação.

Método 2: Clonar Funis ou Sites

Este método é usado para copiar um funil ou site já existente dentro da sua própria conta para outro local ou para duplicar uma página específica.

Para clonar um funil/site inteiro: 1. Na lista de funis ou sites, localize o item desejado e clique nos três pontos (...) ao lado do nome. 2. No menu que aparece, selecione a opção "Clonar". 3. Dê um novo nome para o funil clonado e selecione o local (subconta) de destino (até 3 locais). 4. Clique em "Clonar Funnel" para finalizar.

Para clonar uma etapa ou página específica dentro de um funil: (O artigo original não detalhou os passos específicos para esta ação. Recomenda-se verificar a interface atual da plataforma ou a documentação complementar para instruções precisas.)

Requisitos

  • Acesso de administrador ou usuário com permissões para gerenciar Sites & Funis na agência.
  • Para importação: Um link de compartilhamento válido de um template de funil ou site.
  • Para clonagem: O funil ou site de origem deve já existir em uma das subcontas da sua agência.

FAQ

P1: Como converto um funil em um site, ou vice-versa? * Consulte o artigo dedicado a este tópico para obter instruções detalhadas.

P2: O que acontece se meu link de importação não funcionar? * Se o link não funcionar no momento da tentativa de importação, entre em contato com a equipe de suporte do Full Funnel.

P3: Posso revender esses templates para meus clientes? * Sim. Uma vez adquirido o template, você é livre para utilizá-lo conforme achar melhor, incluindo a revenda para seus próprios clientes.

P4: Com quem entro em contato se precisar de alterações em um template adquirido? * Após a compra, você deve receber um e-mail com as informações de contato do criador do template. Direcione quaisquer dúvidas, solicitações de modificação ou feedback diretamente para ele.

P5: Vocês oferecerão importação de snapshots, automações ou cópias de anúncios no futuro? * Atualmente, o recurso de importação/clonagem via link é suportado apenas para funis e sites. A implementação para outros elementos, como snapshots, automações e cópias de anúncios (FF Ads), está planejada para o futuro.

Adicionar Valores Personalizados a Botões no Construtor de Funis

Como fazer

Para adicionar valores personalizados a botões em um funil, siga estas etapas:

  1. No painel do Full Funnel, acesse Sites e selecione Funnels and Websites.
  2. Clique para editar o funil desejado.
  3. No editor, localize e clique no botão ao qual deseja adicionar um valor personalizado.
  4. No menu de configurações do botão que aparecerá à direita, procure a seção Avançado.
  5. Dentro dessa seção, você encontrará o campo Valor Personalizado.
  6. Digite o valor ou texto personalizado que deseja associar ao botão.
  7. Clique em Salvar no canto superior direito do editor para aplicar a alteração.

Requisitos

  • Você deve ter um funil criado no Full Funnel.
  • É necessário ter permissões de edição para o funil em questão.

FAQ

Para que serve o valor personalizado do botão? O valor personalizado é usado principalmente para integrações e automações. Ele pode ser capturado por gatilhos de workflow (como "Formulário Enviado") e utilizado para segmentar contatos, personalizar comunicações ou passar dados específicos para outras ferramentas conectadas.

Posso usar esse valor em um workflow? Sim. Após configurar o valor, você pode acessar a aba Automações, criar ou editar um workflow e usar um gatilho como "Formulário Enviado" para capturar o valor do botão clicado. Esse valor ficará disponível nas variáveis de merge do contato para uso em ações subsequentes.

O valor personalizado aparece visualmente para o visitante? Não. O valor personalizado é um dado interno (um atributo HTML, como data-value) e não altera o texto visível do botão. Ele é processado em segundo plano quando o botão é clicado.

Como Configurar um Pop-up ao Entrar em um Funil ou Site

Como fazer

Siga estas etapas para configurar um pop-up que aparece automaticamente quando um visitante acessa uma página.

Passo 1: Acessar as Configurações do Pop-up 1. No menu lateral do seu subdomínio, clique em Sites. 2. Na barra de navegação superior, escolha a aba Funis ou Sites. 3. Abra o funil ou site desejado e clique na página específica onde o pop-up deve ser adicionado. 4. Clique no botão azul Editar para abrir o editor de página. 5. Na barra de menu superior do editor, clique no ícone Configurações de Pop-up (⚙️) para abrir o painel de configurações.

Passo 2: Configurar o Gatilho do Pop-up 1. No painel Configurações de Pop-up no lado direito, role para baixo até encontrar o menu suspenso Mostrar pop-up em. 2. Selecione a opção Atraso na lista. 3. Use o controle deslizante abaixo do menu para definir quantos segundos após o carregamento da página o pop-up deve aparecer (ex.: 2 segundos). 4. Clique em Salvar ou Publicar no canto superior direito do editor para aplicar as configurações.

Requisitos

  • Você deve ter um funil ou site criado na plataforma Full Funnel.
  • É necessário ter acesso de edição ao funil ou site.
  • Um pop-up deve estar criado e configurado na página desejada.

FAQ

P: Posso configurar o pop-up para aparecer apenas uma vez por visita? R: Não nativamente com este gatilho. O pop-up configurado com "Atraso" aparecerá sempre que a página for recarregada, a menos que o visitante permaneça na mesma página.

P: E se eu quiser que o pop-up apareça quando alguém tentar sair da página? R: Use a opção Saída no menu suspenso "Mostrar pop-up em". Isso cria um pop-up de intenção de saída, acionado quando o cursor do usuário se move em direção à barra de ferramentas do navegador.

P: Posso usar isso tanto em Funis quanto em Sites? R: Sim! O recurso funciona de forma idêntica para ambos, e o processo de configuração é exatamente o mesmo.

P: O que acontece se o pop-up for aberto manualmente antes do tempo de atraso? R: Se o pop-up for acionado manualmente (por exemplo, por um clique em um botão) antes que o temporizador de atraso termine, o pop-up automático será cancelado. Isso evita que o mesmo pop-up apareça duas vezes durante a mesma visualização de página.

Dashboard: Oportunidades, Valor do Pipeline, Taxa de Conversão, Funil e Distribuição por Estágios

Este artigo explica os principais widgets do dashboard relacionados ao acompanhamento de vendas e pipelines no Full Funnel.

Como os Filtros e Dados Funcionam

  • Filtro de Período: O filtro de data se baseia na Data de Criação dos contatos (leads). Por exemplo, se você filtrar de 24 de janeiro a 25 de julho, o widget "Oportunidades" mostrará todos os contatos que entraram no sistema nesse período e que estão atualmente com status "ABERTO".
  • Visualização por Dia Específico: Para ver o número de contatos que entraram em um dia específico, basta clicar duas vezes na data desejada no gráfico.

Entendendo a Taxa de Conversão

A Taxa de Conversão é calculada automaticamente da seguinte forma: (Número de contatos no status GANHO) / (Número total de contatos que entraram no pipeline)

Exemplo: Se 5 contatos entraram no pipeline e 1 está marcado como GANHO, a taxa de conversão será de 20% (1/5).

Observação Importante: O estágio "GANHO" (WON) é adicionado automaticamente ao final da lista de estágios. Você não precisa criar um estágio manual chamado "GANHO" para rastrear esses contatos.

Interpretando o Gráfico de Funil

O gráfico de Funil mostra a taxa de progressão dos contatos de um estágio para o próximo dentro de um pipeline específico.

  • Como ler: A porcentagem ao lado de cada estágio indica qual percentual dos contatos que entraram no pipeline avançou até aquele ponto.

    • Exemplo: Se 80% aparece ao lado do estágio "Prospectos", significa que 80% dos contatos que ingressaram no pipeline chegaram a esse estágio.

  • Lógica do Cálculo: O sistema constrói o funil como um processo de vendas sequencial. Se um contato está em um estágio posterior, ele é contabilizado como tendo passado por todos os estágios anteriores.

    • Analogia: Imagine um pipeline com os estágios "Ensino Fundamental", "Ensino Médio", "Universidade". Se uma pessoa está na "Universidade", o sistema conta +1 para os estágios "Ensino Fundamental" e "Ensino Médio", presumindo que ela passou por eles.

  • Recomendação de Uso: Para que o gráfico de Funil faça sentido, cada estágio do seu pipeline deve representar um passo mais próximo da venda. Exemplo: Prospecto -> Pesquisa Respondida -> Consulta -> Compra.

  • Estágios que Não se Aplicam: Estágios como "Não Compareceu" (No-Show) ou "Não Comprou" podem distorcer a interpretação do funil, pois não são passos progressivos em direção à venda. Nesses casos, a visualização recomendada é o gráfico de Distribuição por Estágios.

Usando o Gráfico de Distribuição por Estágios

O gráfico de Distribuição por Estágios mostra simplesmente a quantidade de contatos que estão atualmente em cada estágio do pipeline. É a ferramenta ideal para rastrear situações como "Não Compareceu" ou "Compareceu mas Não Comprou".

Dica Avançada - Rastrear Histórico: Se você quiser rastrear por quantos estágios diferentes um contato já passou (ex.: se ele já esteve em "Não Compareceu"), ative a opção Permitir Oportunidade Duplicada nas configurações do pipeline. Em seguida, certifique-se de que a ação de automação "Adicionar/Atualizar Oportunidade" também tenha a opção Permitir oportunidades duplicadas ativada. Isso fará com que um cartão de oportunidade permaneça no estágio sempre que o contato passar por ele.

Requisitos

  • Ter um ou mais pipelines configurados na sua conta.
  • Ter contatos associados a oportunidades dentro desses pipelines para visualizar os dados.

FAQ

1. Por que o gráfico de Funil mostra que 40% dos contatos avançaram para o estágio "Não Compareceu", se só 1 lead de 5 está nesse estágio? Isso acontece porque a lógica do funil presume um caminho progressivo. Se um contato foi marcado como "GANHO", ele é contado como tendo passado por todos os estágios, incluindo o "Não Compareceu". Para visualizar a distribuição real atual, use o gráfico Distribuição por Estágios.

2. Como faço para rastrear quantos contatos já passaram por um estágio específico, como "Não Compareceu"? Ative a configuração Permitir Oportunidade Duplicada no seu pipeline e na ação de automação relevante. Isso criará um registro histórico no estágio cada vez que um contato for movido para ele.

3. Preciso criar um estágio chamado "GANHO"? Não. O estágio "GANHO" é um status do sistema adicionado automaticamente. Você só precisa marcar uma oportunidade como "Ganhou" para que ela seja contabilizada na taxa de conversão.

Como configurar Domínio Raiz ou Subdomínio para seus Funis/Sites

Adicionar um domínio à sua conta permite criar e utilizar funcionalidades de sites e funis. Um domínio é o endereço web, por exemplo, um domínio raiz como meudominio.com ou um subdomínio como www.meudominio.com.

Para começar, você precisa ter um domínio registrado em uma empresa como Cloudflare, GoDaddy ou outras. Depois, você pode integrar o domínio à plataforma Full Funnel para usá-lo.

Importante: Este artigo cobre a configuração manual de um domínio. Para usar o recurso automatizado de conexão de domínio (Domain Connect), consulte a documentação específica.

Como fazer

Para configurar seu novo domínio, siga estas instruções:

Passo 1: Configuração de DNS (no seu Registrador de Domínio) Você deve adicionar um registro A OU um registro CNAME no painel do seu registrador de domínio (ex: Cloudflare, GoDaddy).

  • Método CNAME (Recomendado para subdomínios):

    • Adicione um registro CNAME para seu subdomínio.
    • Campo Host/Nome: Apenas a parte do subdomínio (ex: www para www.meudominio.com, ou blog para blog.meudominio.com).
    • Campo Valor/Destino: sites.ludicrous.cloud
    • Atenção: Não crie um registro CNAME e um registro A para o mesmo host (ex: www). Escolha apenas um método.

  • Método de Registro A (Para domínio raiz ou subdomínio):

    • Adicione um registro A apontando para o IP da Full Funnel.
    • Campo Host/Nome: Para o domínio raiz, use @ ou deixe em branco (depende do registrador). Para um subdomínio, use apenas o nome do subdomínio (ex: www).
    • Campo Valor/IP: 162.159.140.166

Nota sobre Cloudflare: Se estiver usando o Cloudflare, certifique-se de que o Proxy Status do registro esteja definido como "DNS only", pois não suportamos o proxy do Cloudflare para essa configuração.

Após adicionar o registro, pode levar até 24 horas para a propagação das configurações de DNS.

Passo 2: Adicionar o Domínio/Subdomínio à sua Subconta 1. No menu lateral esquerdo, vá em Configurações. 2. Acesse Domínios e Redirecionamentos de URL e clique no botão "+ Conectar um domínio". 3. Clique em "Conectar" para a opção "Funnel/Site/Loja/Blog/Webinar". 4. Digite o Domínio ou Subdomínio e clique em "Continuar". * Para Domínios Raiz (ex: meudominio.com): Digite seu domínio no campo URL do Domínio (use o domínio raiz, sem www). O sistema pode habilitar automaticamente o subdomínio www com um redirecionamento 301. * Para Subdomínios (ex: blog.meudominio.com): Digite o subdomínio completo no campo URL do Domínio. 5. Se a conexão automática (Domain Connect) não for suportada pelo seu registrador, clique no link "Adicionar registro manualmente" para prosseguir.

Passo 3: Associar o Domínio a um Funnel/Site 1. Após a verificação bem-sucedida dos registros DNS, você verá uma tela para escolher qual funil ou site deseja vincular. 2. Selecione o funil ou site desejado. 3. Escolha qual etapa do funil ou página do site será a página de destino padrão para aquele domínio. 4. Clique no botão "Vincular Domínio". O processo será concluído e um certificado SSL será instalado automaticamente, o que pode levar alguns minutos.

Requisitos

  • Uma conta ativa na Full Funnel.
  • Um domínio registrado e gerenciado em um serviço de hospedagem de DNS (ex: GoDaddy, Cloudflare, Namecheap).
  • Acesso administrativo ao painel do seu registrador de domínio para modificar registros DNS (Zona DNS).

FAQ

Posso usar o mesmo Domínio em mais de uma plataforma? Não. Um domínio ou subdomínio só pode apontar para um servidor de cada vez. Você precisa ter apenas um registro DNS (A ou CNAME) configurado para ele.

Posso adicionar o mesmo Domínio a mais de uma conta? Não, um domínio só pode ser verificado e usado em uma única conta/subconta da Full Funnel por vez.

Quantos domínios posso adicionar a uma conta? Consulte seu plano atual. Em caso de dúvida, verifique as informações da sua assinatura ou entre em contato com o suporte.

Preciso comprar SSL separadamente para o Domínio? Não. A Full Funnel fornece e instala automaticamente um certificado SSL gratuito (Let's Encrypt) para cada domínio conectado.

Posso comprar um domínio diretamente com vocês? Atualmente, a Full Funnel não atua como registrador de domínios. Você precisa adquirir seu domínio com um terceiro (GoDaddy, Cloudflare, etc.).

Quantos funis/sites posso conectar a um Domínio? Você pode conectar um funil ou site por domínio/subdomínio. No entanto, dentro de um mesmo funil ou site, você pode usar o mesmo domínio para várias páginas.

Por que meus domínios no Cloudflare não funcionam com Proxy? A Full Funnel não suporta o serviço de proxy (CDN) do Cloudflare nos registros A ou CNAME usados para conectar funis e sites. Você deve definir o status do registro como "DNS only" no painel do Cloudflare.

O que causa o erro "Registro CNAME / A não encontrado."? Este erro ocorre quando a plataforma não consegue encontrar o registro DNS correto apontando seu domínio para sites.ludicrous.cloud (CNAME) ou 162.159.140.166 (A). Verifique: 1. Se você está editando a Zona DNS no provedor correto (onde os nameservers do seu domínio estão apontados). 2. Se o registro foi criado exatamente como descrito no Passo 1. 3. Se não há um registro antigo ou conflitante para o mesmo host/subdomínio. 4. Se você aguardou tempo suficiente para a propagação do DNS (até 24h).

Como remover um Domínio da conta? Vá em Configurações > Domínios e Redirecionamentos de URL. Ao lado do domínio que deseja remover, clique nos três pontos (...) > Excluir e confirme a ação na caixa de diálogo.

Como Criar Categorias no Menu de Navegação

Este artigo mostra como criar menus suspensos (categorias) no menu de navegação do seu site ou funnel, agrupando itens relacionados sob um item principal. Isso ajuda a organizar a navegação e melhora a experiência do usuário com menus mais limpos e intuitivos.

Como fazer

Para criar menus suspensos (categorias), siga estes passos:

  1. Abra o seu funnel ou site no construtor.
  2. Selecione o elemento Menu de Navegação no seu layout.
  3. No painel à direita, clique em Adicionar Item → Novo Item de Menu.
  4. Dê um nome descritivo ao novo item.
  5. Para criar a categoria, arraste e solte o novo item logo abaixo de um item que será o "pai" (por exemplo, "Início").
  6. Quando o item se indentar ligeiramente para a direita, significa que está aninhado e aparecerá como um menu suspenso.
  7. Clique em Salvar e depois em Visualizar. Passe o cursor sobre o item principal para ver o menu suspenso aparecer.

Para visualizar e organizar os itens do menu, você pode: * Usar a visualização para verificar a estrutura. * Passar o cursor sobre qualquer item principal para confirmar a funcionalidade do menu suspenso. * Usar a interface de arrastar e soltar para reordenar ou renomear os itens. * Aninhar vários itens sob o mesmo item principal.

Requisitos

  • Acesso ao construtor de funnels ou sites do Full Funnel.
  • Um elemento de Menu de Navegação já adicionado à sua página.

FAQ

P: Posso criar mais de um nível de menus suspensos (submenus)? R: Não. Para manter a navegação amigável, especialmente em dispositivos móveis, o sistema é limitado a uma camada de páginas aninhadas.

P: Os itens do menu podem vincular a seções dentro da mesma página? R: Sim. Você pode usar links âncora para rolar até seções específicas.

P: Como removo ou renomeio itens do menu? R: Clique nos 3 pontos ao lado do item na barra lateral. Edite o nome ou clique no botão Excluir para remover.

P: Este processo é o mesmo para funnels e sites? R: Sim, o elemento Menu de Navegação funciona da mesma forma em ambos.

Como Impedir que Mecanismos de Busca Indexem sua Página

Você pode usar uma meta tag personalizada para instruir os robôs de busca (como Google, Bing, etc.) a não indexarem uma página específica do seu site ou funil no Full Funnel. Isso é útil para páginas privadas, de teste ou em construção.

Como fazer

Para adicionar a meta tag que bloqueia a indexação, siga estes passos no editor do seu site ou funil:

  1. No editor, clique no botão "Configurações" (ícone de engrenagem) no canto inferior esquerdo.
  2. No menu que abre, selecione a opção "SEO".
  3. Role até a seção "Meta Tags Personalizadas".
  4. Clique no botão "+ Adicionar Meta Tag".
  5. No campo "Nome", insira robots.
  6. No campo "Conteúdo", insira noindex.
  7. Clique em "Salvar" para aplicar a alteração à página.

A tag será adicionada ao código da sua página no seguinte formato:

<meta name="robots" content="noindex">

Requisitos

  • Acesso de edição ao site ou funil no Full Funnel.
  • A meta tag deve ser aplicada individualmente a cada página que você deseja bloquear da indexação.

FAQ

P: Isso impede 100% que a página apareça nos resultados de busca? R: A tag noindex é uma diretiva respeitada pela maioria dos mecanismos de busca reputáveis. No entanto, é uma solicitação, não uma barreira técnica. O tempo para a página ser removida dos índices de busca varia.

P: Posso bloquear apenas o Google, permitindo outros buscadores? R: Sim. Em vez de usar robots, use googlebot no campo "Nome" e noindex no campo "Conteúdo". A tag resultante <meta name="googlebot" content="noindex"> instrui especificamente o robô do Google a não indexar a página.

P: A tag afeta o rastreamento (crawling) ou apenas a indexação? R: A diretiva noindex instrui o mecanismo de busca a não incluir a página em seus resultados. Ela não impede necessariamente que o robô rastreie a página. Para controlar o rastreamento, seria necessária uma diretiva diferente no arquivo robots.txt.

Como Ocultar ou Exibir Elementos em Funis Usando um Temporizador

Como fazer

Para controlar a visibilidade de elementos em seus funis com base em um tempo específico, siga estas etapas:

  1. Acesse o Editor de Funnel:

    • No painel do Full Funnel, vá até Sites e selecione o funil que deseja editar.
    • Clique em "Editar Funnel" para abrir o editor visual.

  2. Adicione ou Selecione o Elemento:

    • No menu lateral esquerdo do editor, arraste e solte o elemento que deseja controlar (como um botão, imagem ou texto) na página.
    • Se o elemento já existir, clique sobre ele para selecioná-lo.

  3. Configure a Ação de Temporizador:

    • Com o elemento selecionado, vá para a aba "Configurações" no painel lateral direito.
    • Localize a seção "Ações" ou "Interatividade".
    • Clique em "Adicionar Ação" ou opção similar.
    • Selecione a ação "Ocultar" ou "Exibir".
    • No campo de configuração da ação, procure pela opção de "Atraso" ou "Temporizador".
    • Defina o tempo (em segundos ou minutos) após o qual a ação deve ocorrer. Por exemplo, para exibir um pop-up após 30 segundos, defina o atraso para 30 segundos.

  4. Defina a Condição Inicial (Opcional):

    • Para um elemento que deve ficar visível inicialmente e depois sumir, adicione uma ação "Ocultar" com atraso.
    • Para um elemento que deve ficar oculto inicialmente e depois aparecer, você pode precisar marcar a opção "Oculto" nas configurações gerais do elemento e depois adicionar uma ação "Exibir" com atraso.

  5. Salve e Publique:

    • Clique em "Salvar" no canto superior direito do editor.
    • Em seguida, clique em "Publicar" para que as alterações entrem em vigor no seu funil ao vivo.

Requisitos

  • Acesso a um funil criado no Full Funnel.
  • Permissões para editar o funil.
  • O elemento que você deseja controlar (botão, imagem, bloco de texto, etc.) deve estar na página.

FAQ

P: Posso usar isso para criar um pop-up que aparece após um tempo específico? R: Sim. Basta adicionar o elemento de pop-up à página, garantir que ele esteja oculto inicialmente (nas configurações do elemento) e então adicionar uma ação "Exibir" com o atraso desejado.

P: É possível combinar múltiplas ações de temporizador em um único elemento? R: Sim. Você pode adicionar várias ações a um elemento. Por exemplo, fazer um botão aparecer após 10 segundos e desaparecer novamente após 30 segundos.

P: O temporizador começa a contar a partir de quando? R: O temporizador geralmente inicia a contagem a partir do momento em que a página do funil é completamente carregada no navegador do visitante.

P: Posso usar essa funcionalidade em sites comuns também, ou apenas em funis? R: Esta funcionalidade está disponível tanto para páginas de funis quanto para sites criados no construtor do Full Funnel.

Como Usar Parâmetros de URL em Sites e Funis

Como fazer

Os parâmetros de URL (também conhecidos como query strings) permitem passar informações dinâmicas para suas páginas de funil ou site no Full Funnel. Eles são adicionados ao final do URL da sua página.

Para capturar e usar um parâmetro em seu funil ou site:

  1. Acesse o Editor: No painel do Full Funnel, vá até Sites & Funnels e abra o funil ou site desejado no editor.
  2. Adicione um Elemento Dinâmico: Arraste e solte um elemento de texto (como "Paragraph" ou "Heading") na sua página.
  3. Insira o Código Dinâmico: Clique duas vezes no elemento de texto para editar. No menu de formatação, clique no ícone </> (código) para inserir um código dinâmico.
  4. Use a Sintaxe Correta: Digite o seguinte código, substituindo NOME_DO_PARAMETRO pelo nome do parâmetro que você deseja capturar (por exemplo, nome, utm_source, id): {{query.NOME_DO_PARAMETRO}}
  5. Exemplo Prático: Se o URL for seusite.com/landingpage?nome=Joao, use o código {{query.nome}}. O texto exibido na página será "Joao".

Para criar um link com parâmetros: Ao adicionar um botão ou link, no campo de URL, você pode adicionar os parâmetros manualmente após o endereço principal, como: https://seusite.com/obrigado?origem=facebook&cliente=sim.

Requisitos

  • Uma conta ativa no Full Funnel.
  • Um funil ou site publicado.
  • Conhecimento básico da estrutura de um URL (ex: ?chave=valor&outra_chave=outro_valor).

FAQ

P: Os parâmetros não estão aparecendo na minha página. O que fazer? R: Verifique se: 1. O nome do parâmetro no código {{query.XXX}} está escrito exatamente como no URL (case-sensitive). 2. O funil ou site foi publicado após a adição do código dinâmico. 3. Você está acessando a URL completa com os parâmetros (ex: seusite.com/pagina?parametro=valor).

P: Posso usar esses parâmetros para preencher formulários automaticamente? R: Sim. Na configuração de um campo de formulário, você pode definir um Valor Padrão e usar a mesma sintaxe {{query.parametro}} para preenchê-lo automaticamente quando o parâmetro estiver presente no URL.

P: Esses parâmetros são passados entre as páginas do meu funil? R: Não automaticamente. Se você precisa que um parâmetro capturado na Página A esteja disponível na Página B, deve incluí-lo manualmente nos links ou botões que direcionam para a próxima etapa (ex: no botão "Próximo", definir a URL como pagina-b?mesmo_parametro={{query.parametro}}).

Como Criar e Adicionar Produtos a um Funil ou Site (Pagamentos)

Como fazer

Para gerenciar seus produtos e adicioná-los a um funil ou site, siga estes passos:

  1. Acesse a seção Pagamentos no menu principal da plataforma Full Funnel.
  2. Clique em Produtos para gerenciar seu catálogo.
  3. Para criar um novo produto, clique no botão "Criar Produto".
  4. Preencha os detalhes do produto:
    • Nome: Dê um nome claro para o produto.
    • Descrição: Adicione uma descrição detalhada.
    • Preço: Defina o valor. Você pode criar produtos de pagamento único ou assinatura (recorrente).
    • Imagem: Faça o upload de uma imagem do produto.
  5. Após salvar o produto, vá até o editor do seu Funil ou Site.
  6. Na página onde deseja adicionar o formulário de pedido, localize e adicione o elemento "Formulário de Pedido".
  7. Ao configurar o formulário, você poderá selecionar os produtos criados na seção Pagamentos para venda.

Para Sites: Se você não visualizar a opção de adicionar um produto diretamente, clique nos três pontos (⋯) na página do site onde o formulário de compra deve aparecer. A opção de adicionar o formulário de pedido e selecionar os produtos estará disponível ali.

Requisitos

  • Você precisa ter permissão de acesso à seção Pagamentos e ao editor de Funis/Sites.
  • É necessário ter um funil ou site criado para adicionar o produto.

FAQ

P: Estou usando um Site e não vejo como adicionar um produto. O que fazer? R: Para adicionar um produto a um site, clique nos três pontos (⋯) localizados na página do site onde o seu formulário de pedido ficará. A opção de configurar o formulário e selecionar os produtos estará disponível nesse menu.
