Calendários de Grupo

Calendários de Grupo permitem combinar múltiplos calendários individuais sob um único link, oferecendo agendamentos flexíveis.

Importante: Calendários de grupo não criam calendários individuais automaticamente. Você precisa criar os calendários individuais antes de adicioná-los a um grupo.

Benefícios Principais

  • Combine múltiplos calendários em um único link de agendamento.
  • Permita que clientes escolham entre membros da equipe ou serviços.
  • Organize links de calendário por departamento, disponibilidade ou serviços.
  • Reduza a desordem, substituindo múltiplos links de calendário individuais.

Como Criar um Calendário de Grupo

  1. Vá em Configurações → Calendários.
  2. Clique em "Criar Grupo".
  3. Preencha os seguintes campos:
    • Nome do Grupo: Insira um nome para o grupo de calendários.
    • Descrição: Adicione uma breve descrição (opcional).
    • Modelo: Escolha entre os modelos "Clássico" ou "Neo" para a visualização do grupo.
    • URL do Grupo: Crie uma URL com um slug apropriado.
  4. Clique em "Criar".
  5. Na janela pop-up que aparece, você tem duas opções:
    • Escolher Calendários: Selecione os calendários que você deseja adicionar ao grupo diretamente na janela pop-up.
    • Farei Isso Mais Tarde: Ignore esta etapa e adicione os calendários ao grupo posteriormente.
  6. Se você clicou em "Escolher Calendários", selecione os calendários desejados.
  7. Clique em "Confirmar".

Observação: Se nenhum calendário aparecer, certifique-se de ter criado calendários individuais previamente.

Sincronizando com Calendários Externos

Certifique-se de que cada membro da equipe tenha um "Calendário Principal" configurado nas configurações do calendário. Isso é necessário para sincronizar agendamentos com calendários de terceiros, como Google e Outlook.

Perguntas Frequentes

P: Posso criar um calendário diretamente na tela de configuração do Calendário de Grupo? R: Não. Você deve primeiro criar calendários individuais na aba "Calendários". Depois de criados, você pode atribuí-los a um Calendário de Grupo. Calendários de Grupo são contêineres e não criam ou configuram calendários individuais por conta própria.

P: Posso agrupar calendários de Classe ou Coletivos? R: Não. Calendários de Grupo são separados dos tipos de calendário de Classe e Coletivo. Cada um tem sua própria estrutura e comportamento. Você pode usar fluxos de agendamento separados para esses tipos de calendário com base no seu caso de uso.

P: O que acontece se eu excluir um calendário dentro de um grupo? R: O calendário de grupo ainda existirá, mas o calendário excluído não aparecerá mais como uma opção de agendamento. Sempre verifique a composição do grupo após editar ou excluir calendários.

Calendários de Grupo

Calendários de Grupo permitem agrupar múltiplos calendários individuais sob um único link, oferecendo agendamentos mais flexíveis.

Como criar um Calendário de Grupo

  1. Vá em Configurações → Calendários.
  2. Clique em "Criar Grupo".
  3. Insira um Nome para o Grupo e uma descrição (opcional).
  4. Selecione um template (Clássico ou Neo) para a visualização do grupo.
  5. Crie uma URL para o Grupo, definindo um Slug apropriado.
  6. Clique em "Criar".
  7. Na janela que aparece, escolha uma das opções:
  8. "Escolher Calendários": Selecione os calendários que você quer adicionar ao grupo.
  9. "Farei Isso Mais Tarde": Adicione os calendários ao grupo posteriormente.
  10. Se escolheu "Escolher Calendários", selecione os calendários desejados e clique em "Confirmar".

Como Localizar o ID de um Calendário de Grupo

Este artigo mostra como encontrar o ID de qualquer Calendário de Grupo na nossa plataforma, um passo fundamental para integrações, automações e uso da API.

O que é um ID de Calendário de Grupo?

Um ID de Calendário de Grupo na nossa plataforma é um identificador único atribuído a cada calendário de grupo que você cria. Ele é usado em integrações, Automações e chamadas de API para referenciar aquele calendário específico. Este ID garante que eventos, automações e agendamentos estejam ligados ao calendário de grupo correto, especialmente ao trabalhar com múltiplos membros de equipe ou serviços.

Quando um ID de Calendário é Necessário?

Você pode precisar do ID do Calendário para os seguintes casos de uso:

  • Acionar Automações baseadas em agendamentos de calendário.
  • Enviar dados via Webhooks ou chamadas de API.
  • Integrar com ferramentas externas (ex: Zapier, Pabbly, Make).
  • Cenários avançados de resolução de problemas e suporte.

Como Localizar o ID de um Calendário de Grupo Específico

  1. Navegue até Grupos de Calendários:
    • Vá para sua subconta na nossa plataforma e clique em Configurações no menu esquerdo.
    • Clique em Calendários e selecione Grupos sob Configurações do Calendário.
  2. Localizando e Copiando o ID do Calendário de Grupo:
    • Na lista de Calendários de Grupo, clique no ícone Compartilhar Grupo do grupo desejado.
    • Clique em Link de Agendamento, então selecione e copie a última parte do Link Permanente.
    • Ou você pode clicar em Código de Incorporação, então selecione e copie a parte do ID antes do sublinhado.

Perguntas Frequentes

P: Posso usar o mesmo ID de Calendário em múltiplas Automações?

R: Sim! Desde que as Automações estejam conectadas ao mesmo Calendário de Grupo, o ID do Calendário pode ser reutilizado em diferentes Automações e integrações.

P: Os IDs de Calendário são os mesmos para todos os usuários na subconta?

R: Sim. O ID do Calendário está ligado ao próprio calendário e permanece o mesmo independentemente do usuário que o acessa, desde que ele tenha permissão.

P: O ID do Calendário mudará se eu renomear o calendário?

R: Não, renomear o calendário não afetará o ID do Calendário. O ID permanece constante independentemente do nome de exibição do calendário.

P: E se o formato do ID do Calendário mudar?

R: Nossa plataforma notificará os usuários sobre quaisquer mudanças nas estruturas de calendário ou formatos de URL. Marque este artigo para se manter atualizado.

P: E se eu excluir um calendário? O ID ainda funcionará?

R: Não, uma vez que um calendário é excluído, seu ID se torna inválido. Quaisquer referências a esse ID em Automações ou integrações deixarão de funcionar.

P: Posso usar IDs de Calendário para filtrar agendamentos na API?

R: Sim, se você estiver extraindo dados da API, você pode usar o ID do Calendário para filtrar agendamentos ou dados de compromissos para um calendário específico.
