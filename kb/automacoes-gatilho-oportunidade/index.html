O que é o Trigger "Oportunidade Alterada" em Automações?

O trigger "Oportunidade Alterada" automatiza ações quando um registro de oportunidade é atualizado. Diferente do trigger "Status da Oportunidade Alterado", que rastreia apenas atualizações de status, este trigger responde a modificações como valor do lead, etapa do funil, atribuição ou campos personalizados.

Benefícios

  • Rastreamento Abrangente: Automatize ações com base em qualquer alteração em uma oportunidade, não apenas atualizações de status.
  • Notificações em Tempo Real: Notifique membros da equipe imediatamente quando uma oportunidade é atualizada.
  • Precisão Aprimorada do CRM: Garanta a manutenção automatizada de registros e a consistência dos dados.
  • Melhor Gestão do Funil de Vendas: Acione acompanhamentos ou atribuições de tarefas quando as oportunidades são modificadas.
  • Automação Personalizável: Aplique filtros de automação para adaptar a automação a alterações específicas de oportunidades.
  • Operadores de Filtro Flexíveis: Use operadores como "Foi Alterado", "Foi Alterado Para" e "Igual a" para controlar precisamente quando os fluxos de trabalho são acionados.

Como configurar o Trigger "Oportunidade Alterada"

  1. Acesse o construtor de Automações:
  2. Faça login na nossa plataforma, vá para Automações > Automações.
  3. Crie uma nova automação ou abra uma existente.
  4. Adicione o trigger "Oportunidade Alterada":
  5. Clique em + Adicionar Novo Trigger.
  6. Pesquise por "Oportunidade Alterada" na lista de triggers disponíveis e selecione-o.
  7. Nomeie o Trigger:
  8. Dê ao seu trigger um nome claro e descritivo que reflita seu propósito.
  9. Aplicando Filtros:
  10. Os filtros ajudam a ajustar seu fluxo de trabalho para que ele seja acionado apenas quando mudanças específicas acontecerem em uma oportunidade.

Filtro "Atribuído A"

O filtro "Atribuído A" aciona o fluxo de trabalho quando uma oportunidade é reatribuída a um membro diferente da equipe. - Operadores disponíveis: - Foi Alterado: Aciona quando o usuário atribuído muda para qualquer outro usuário. - Foi Alterado Para: Aciona somente quando a oportunidade é atribuída a um usuário específico selecionado. - Igual a: Aciona quando qualquer alteração ocorre em oportunidades atualmente atribuídas a um usuário específico.

Filtro de Tag (antigo "Tem Tag")

Use filtros de Tag para controlar quando "Oportunidade Alterada" deve ser acionado com base nas tags de oportunidade. - O filtro de Tag permite acionar um fluxo de trabalho com base nas tags atribuídas a uma oportunidade.

Filtro "No Funil"

O filtro "No Funil" aciona o fluxo de trabalho quando uma oportunidade passa por uma mudança dentro de um funil específico. - Operadores disponíveis: - Foi Alterado: Aciona quando a oportunidade se move para qualquer funil diferente. - Foi Alterado Para: Aciona somente quando a oportunidade se move para um funil específico selecionado. - Igual a: Aciona quando qualquer alteração ocorre em oportunidades dentro de um funil específico.

Filtro "Valor do Lead"

O filtro "Valor do Lead" aciona um fluxo de trabalho quando o valor monetário de uma oportunidade aumenta ou diminui.

Filtro "Motivo da Perda"

O filtro "Motivo da Perda" permite automatizar fluxos de trabalho quando uma oportunidade é marcada como Perdida e um motivo é atribuído. - Operadores disponíveis: - Foi Alterado: Aciona o fluxo de trabalho sempre que o motivo da perda é atualizado, independentemente do valor anterior. - Foi Alterado Para: Aciona o fluxo de trabalho somente quando o motivo da perda é atualizado para um valor específico. - Igual a: Aciona quando qualquer alteração ocorre em oportunidades que têm um motivo de perda específico atribuído.

Filtro de Status

O filtro "Status" permite que os fluxos de trabalho sejam acionados com base no status atual de uma oportunidade. - Operadores disponíveis: - É: Aciona o fluxo de trabalho quando uma oportunidade corresponde ao status selecionado. - Não é: Aciona o fluxo de trabalho quando uma oportunidade não corresponde ao status selecionado. - Foi Alterado: Aciona quando o status muda para qualquer outro status. - Foi Alterado Para: Aciona quando o status muda para um status específico.

Perguntas Frequentes

P: Qual a diferença entre o trigger "Oportunidade Alterada" e "Status da Oportunidade Alterado"? R: O trigger "Oportunidade Alterada" responde a qualquer modificação na oportunidade (valor, etapa, atribuição, etc.), enquanto "Status da Oportunidade Alterado" rastreia apenas mudanças de status.

Gatilho: Oportunidade Criada

Este gatilho inicia uma automação quando uma nova oportunidade é criada no funil de vendas.

Como configurar

  1. Dentro do editor de Automações, adicione um novo gatilho.
  2. Selecione "Oportunidade Criada" na lista de gatilhos disponíveis.
  3. Configure os filtros (opcional):
    • Valores Personalizados: Adicione filtros baseados em valores personalizados da oportunidade.

Requisitos

  • Ter um funil de vendas configurado em Oportunidades.
  • Campos personalizados devem estar previamente criados, se for usá-los nos filtros.
