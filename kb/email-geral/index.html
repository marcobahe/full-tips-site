Personalizar Mensagens de Solicitação de Avaliação (SMS/E-mail)

Como fazer

Para configurar mensagens personalizadas de solicitação de avaliação, siga estes passos:

Passo 1: Configurar o Link de Avaliação 1. No menu lateral esquerdo, clique em Reputação. 2. Vá para a aba Configurações no topo. 3. Na seção Link de Avaliação, você tem duas opções: * Balanceamento de Avaliações: Ative a opção para distribuir os clientes entre as plataformas de avaliação conectadas (como Google). * Link Personalizado: Selecione esta opção e cole um URL específico para enviar todos os clientes para uma única página.

Passo 2: Ativar Solicitações por SMS 1. Acesse Reputação → Configurações → Solicitações por SMS. 2. Ative a chave no canto superior direito. 3. Configure os controles: * Quando enviar SMS após o check-in? Escolha o atraso para o primeiro envio. * Até ser clicado, repetir a cada: Defina o intervalo para as tentativas de reenvio. * Máximo de tentativas: Escolha quantas mensagens de follow-up enviar. 4. Clique em Criar Novo para criar um modelo de SMS ou use o ícone de lápis para editar um existente.

Passo 3: Criar ou Editar um Modelo de SMS 1. Ao clicar em Criar Novo, escolha Criar Novo do Zero ou Selecionar dos Modelos Pré-criados. 2. Dê um Nome ao Modelo. 3. (Opcional) Ative Solicitar com Imagem para adicionar uma imagem. 4. Escreva sua mensagem. Inclua o token {{reputation.review_link}} para o link de avaliação. 5. Clique em Salvar.

Passo 4: Ativar Solicitações por E-mail 1. Acesse Reputação → Configurações → Solicitações por E-mail. 2. Ative a chave no canto superior direito. 3. Configure os controles de tempo e tentativas, semelhante ao SMS.

Passo 5: Criar ou Importar um Modelo de E-mail 1. Clique em Criar Novo e escolha Criar novo Modelo ou Importar da Biblioteca de Modelos. 2. No editor de e-mail, use a barra lateral esquerda para arrastar elementos (texto, imagens, botões) para o layout. 3. Arraste o elemento "Link de Avaliação" para o seu e-mail. Este botão usará automaticamente o URL configurado no Passo 1. 4. Clique em Salvar no canto superior direito.

Passo 6: Definir a Sequência de Envio (Modelos "Ao Vivo" e de "Nova Tentativa") * Para E-mail: Clique em Definir Modelos de E-mail. Selecione o modelo Ao Vivo (envio inicial) e atribua modelos para cada slot de Nova Tentativa. Clique em Salvar. * Para SMS: Clique em Definir Modelos de SMS e faça a mesma seleção para a sequência de SMS.

Requisitos

  • Acesso ao menu Reputação e à aba Configurações.
  • Para usar Balanceamento de Avaliações e direcionar para o Google, é necessário ter um Google Meu Negócio conectado nas Integrações.
  • Para enviar solicitações, você precisa ter modelos de SMS e/ou E-mail criados e atribuídos na sequência.

FAQ

P: O que acontece depois que um cliente clica no link de avaliação? R: O sistema para de enviar novas tentativas (follow-ups) por aquele canal específico (SMS ou E-mail) para aquele contato.

P: Qual a diferença entre Balanceamento de Avaliações e Link Personalizado? R: O Balanceamento distribui os clientes entre diferentes sites de avaliação conectados. O Link Personalizado envia todos os clientes para uma única URL de sua escolha.

P: Qual token devo usar no SMS para o link de avaliação? R: Use sempre {{reputation.review_link}}. Ele será automaticamente preenchido com o link que você configurou em Reputação → Configurações → Link de Avaliação.

P: Posso automatizar o envio de solicitações? R: Sim. Use a ação Solicitar Avaliação em uma Automação (Workflow) para acionar o envio automaticamente, por exemplo, após uma mudança no status de um agendamento.

P: Adicionar uma imagem a um SMS muda o custo? R: Pode mudar. Uma mensagem com imagem pode ser enviada como MMS, que pode ter tarifas diferentes das SMS. Consulte as taxas do seu provedor de mensagens.

P: Preciso ter o Google Meu Negócio conectado? R: Apenas se quiser usar o Balanceamento de Avaliações para direcionar clientes ao Google. Se preferir usar outro site ou sua própria página, configure um Link Personalizado.

Uma Introdução à Entregabilidade de E-mail

O que é entregabilidade e por que é importante?

Entregabilidade é a medida de quantos dos seus e-mails chegam à caixa de entrada dos seus assinantes, em vez de irem para a pasta de spam. Depois de tanto trabalho no seu marketing por e-mail, você quer ter certeza de que suas mensagens estão realmente sendo vistas! É por isso que a entregabilidade é crucial.

Como a entregabilidade é determinada?

Muitos fatores influenciam a colocação na caixa de entrada. Em geral, os provedores de e-mail (como Gmail, Outlook, etc.) tentam determinar três coisas: 1. A mensagem é segura? 2. A mensagem é desejada pela maioria dos assinantes? 3. A mensagem é desejada por este assinante específico?

Se a resposta para todas as três perguntas for "sim", a mensagem pertence à caixa de entrada. Se alguma resposta for "não", a mensagem vai para o spam. Para tomar essa decisão, os provedores geralmente analisam a reputação do remetente que calcularam para você.

Reputação do Remetente

Provedores de e-mail criam uma pontuação de reputação para o seu domínio de envio (ex: @suaempresa.com). Pense nisso como uma pontuação de crédito, mas infelizmente não há uma fonte única que mostre exatamente qual é sua pontuação e por quê. Os provedores mantêm essa pontuação e os algoritmos usados para determiná-la em sigilo. No entanto, o fator mais importante é a forma como os assinantes interagem (ou não) com suas mensagens.

Engajamento do Assinante

A melhor maneira dos provedores determinarem onde seu e-mail pertence é vendo como seus assinantes interagem com ele. Existem ações que seus assinantes podem tomar que aumentam sua reputação e ações que a diminuem.

Recomendações para promover um engajamento positivo:

1. Saúde da Lista * Permissão Direta: Só importe assinantes que lhe deram permissão direta para receber e-mails de marketing. Qualidade é muito mais importante que quantidade. * Formulários Seguros: Proteja seus formulários com confirmação dupla (duplo opt-in) e/ou ReCaptcha. Formulários não seguros podem ser abusados por bots, o que prejudica sua entregabilidade. * Limpeza Regular: Não deixe assinantes inativos se acumularem. Limpe sua lista regularmente para que assinantes não engajados não se tornem a maioria, o que derrubaria sua reputação.

2. Consistência Mudanças bruscas no padrão de envio são vistas como arriscadas. Mantenha consistência em: * Domínio de Envio: Envie de um domínio que os assinantes reconheçam. Evite enviar de um domínio novo ou que não envia e-mails há muito tempo (mais de 6 meses). * Volume: Envie para um número consistente de assinantes por semana. Evite picos bruscos (ex: de 5.000 para 20.000 da noite para o dia). Se precisar adicionar muitos novos assinantes, comece enviando em pequenos lotes. * Frequência: Envie pelo menos uma vez por mês para manter sua reputação estabelecida. Não envie com frequência excessiva para não causar irritação. Pergunte aos assinantes com que frequência desejam ouvir de você e permita que eles ajustem suas preferências.

3. Conteúdo Se sua reputação não estiver bem estabelecida, o conteúdo pode influenciar a entregabilidade. * Evite Encurtadores de Link: São muito usados por spammers e podem levantar suspeitas. Seja transparente e use links diretos. * Evite Linguagem Promocional Exagerada: Mesmo vendendo, escreva de forma humana e conversacional. Evite palavras e frases típicas de spam que soam urgentes ou sensacionalistas. * Proporção Imagem/Texto: Não há um número mágico, mas garanta que haveria texto suficiente para comunicar sua mensagem se as imagens não carregarem. * Evite Truques: Subject lines como "Re: Seu Pedido #2095642" podem dar um pico rápido em aberturas, mas fazem você perder a confiança do assinante, aumentando as chances de marcação como spam. * Incentive Engajamento Autêntico: Peça para os assinantes responderem de forma genuína (ex: "Responda com sua música favorita no momento"). Isso ajuda a construir relacionamento e reputação.

Resumo

Uma boa entregabilidade vem de colocar os assinantes em primeiro lugar: construir uma lista limpa e engajada, enviar com consistência e criar conteúdo que os assinantes valorizam. Isso ajuda os provedores a confiarem em você e entregarem seus e-mails nas caixas de entrada.

FAQ (Perguntas Frequentes)

Quais ferramentas posso usar para monitorar minha entregabilidade? Ferramentas como Mailgun, SendGrid e Litmus podem ajudar. Elas fornecem insights sobre taxas de bounce, aberturas e reputação do remetente, permitindo otimizar suas campanhas.

Como posso me recuperar de uma lista negra (blacklist)? Identifique a causa (ex: reclamações de spam, más práticas de envio), corrija os problemas, garanta conformidade com as melhores práticas e siga o processo específico de remoção da lista negra em questão. Reconstruir uma reputação positiva leva tempo.

Qual o papel do engajamento na entregabilidade? Métricas de engajamento (taxas de abertura e clique) influenciam muito a entregabilidade. Um engajamento alto sinaliza para os provedores que seu conteúdo é valioso, melhorando sua reputação. Um engajamento baixo pode fazer seus e-mails serem filtrados para o spam.

Erro ao Enviar E-mail de Teste na Etapa de E-mail da Campanha

Como Resolver

Se você encontrar a mensagem de erro "An error occurred, please try again" ao tentar enviar um e-mail de teste em uma campanha, ou se a ação de gatilho "Enviar E-mail" não estiver sendo acionada, siga estas etapas de verificação:

  1. Verifique o Conteúdo do E-mail/SMS:

    • Revise o texto do seu e-mail ou SMS.
    • Certifique-se de que não faltam chaves de fechamento }.
    • Verifique se não está usando excessivamente três chaves {{{ de forma incorreta. O uso de variáveis personalizadas geralmente segue o formato {{NomedoCampo}}.

  2. Valide os Valores Personalizados (Merge Tags):

    • Confirme se todos os campos personalizados inseridos no e-mail (como {{Nome}}, {{Link}}) estão escritos corretamente e existem para o contato.
    • Atenção especial aos links: Garanta que os links personalizados usem valores válidos. Consulte a lista oficial de valores personalizados suportados na sua conta do Full Funnel.

  3. Confira os Endereços de E-mail:

    • Verifique se o endereço de e-mail do destinatário (incluindo o de teste) está escrito corretamente.
    • Procure por erros comuns, como usar uma vírgula no lugar do ponto (ex.: test@gmail,com). O correto é test@gmail.com.

  4. Formato do Campo "De" (Remetente):

    • O campo "De" deve estar no formato de um endereço de e-mail válido.
    • Não use apenas um nome. Por exemplo, em vez de "ABC Company", o campo deve conter um e-mail como info@abccompany.com.

Requisitos

  • Acesso de edição à campanha ou automação onde o e-mail está configurado.
  • Conhecimento básico da sintaxe de variáveis personalizadas (Merge Tags) do Full Funnel.
  • Um endereço de e-mail de teste válido para enviar a simulação.

FAQ

P: O que causa o erro de chaves faltantes ou em excesso? R: Isso geralmente acontece quando uma variável personalizada (ex.: {{Nome}) não é fechada corretamente com duas chaves, ou quando a sintaxe para funções mais avançadas é usada de forma errada.

P: Como encontro a lista de valores personalizados válidos para links? R: Dentro do editor de e-mail do Full Funnel, normalmente há um botão ou menu para inserir "Campos Personalizados". Lá você encontrará a lista oficial. Não use links criados manualmente com {{{ sem seguir a documentação específica.

P: Meu e-mail de teste foi enviado, mas o e-mail automático da campanha não dispara para os contatos. É o mesmo problema? R: Sim, as mesmas verificações se aplicam. Se a ação "Enviar E-mail" em uma automação não está sendo acionada, os itens mais comuns a checar são o formato do e-mail do remetente e a validade das variáveis personalizadas no corpo da mensagem.

Quais Estatísticas de Entregabilidade de E-mail Devo Observar?

Como fazer

Para monitorar a saúde do seu envio de e-mails no Full Funnel, você deve analisar regularmente as seguintes métricas principais. Aqui está um guia passo a passo:

  1. Acesse os Relatórios de E-mail:

    • No menu lateral esquerdo da plataforma, clique em Relatórios.
    • Navegue até a seção de relatórios de E-mail ou Campanhas.
    • Selecione a campanha ou o domínio específico que deseja analisar.

  2. Analise as Métricas-Chave:

    • Taxa de Entregue: Verifique a porcentagem de e-mails que chegaram ao servidor do destinatário.
      • ✅ Bom: 98% ou mais.
      • ⚠️ Atenção: Entre 97% e 98%.
      • ❌ Crítico: Abaixo de 97%. Isso indica que mais de 3% dos e-mails não foram entregues, um sinal de alerta.
    • Taxa de Abertura: Verifique quantos e-mails entregues foram abertos.
      • ✅ Bom: Almeje taxas nos anos 40, 50 ou 60%, dependendo do seu setor e público.
      • ⚠️ Média: A média do setor varia entre 16% e 26%.
      • ❌ Crítico: Abaixo de 15%.
    • Taxa de Cliques: Verifique quantos destinatários clicaram em links dentro do e-mail.
      • ✅ Bom: Almeje acima de 10%.
      • ⚠️ Média: A média do setor é de 7% a 9%.
      • ❌ Crítico: Abaixo de 5%.
    • Taxa de Resposta: Para campanhas que buscam uma resposta, verifique essa métrica.
      • ✅ Bom: Almeje 30% ou mais.
      • ⚠️ Média: A média do setor é de 15% a 25%.
      • ❌ Crítico: Abaixo de 10%.
    • Taxa de Reclamações (Spam): Monitore quantos destinatários marcaram seu e-mail como spam.
      • ✅ Bom: O mais próximo possível de 0,00%.
      • ⚠️ Atenção: Acima de 0,04% pode começar a afetar sua reputação.
      • ❌ Crítico: Taxas altas podem levar a bloqueios.
    • Taxa de Bounce (Falha): Monitore e-mails que não puderam ser entregues.
      • ✅ Bom: Bem abaixo de 1%.
      • ⚠️ Atenção: Acima de 2% a 2,5% é preocupante.
      • ❌ Crítico: Taxas altas prejudicam sua reputação.

  3. Monitore Regularmente e Ajuste:

    • Verifique essas estatísticas semanalmente.
    • Compare o desempenho no nível do domínio e no nível de campanhas individuais.
    • Se identificar problemas (como taxa de entregue baixa ou reclamações altas), revise suas práticas: verifique a higiene da sua lista, o conteúdo do e-mail e a segmentação do público.

Requisitos

  • Acesso à sua conta do Full Funnel.
  • Campanhas de e-mail já enviadas para gerar dados.
  • Permissões para visualizar relatórios na plataforma.

FAQ

O que significa "Entregue" exatamente? "Entregue" significa que o e-mail chegou ao servidor de e-mail do destinatário. Não é uma garantia de que ele entrou na Caixa de Entrada principal; pode ter sido para Spam ou Promoções.

Por que minha taxa de abertura pode estar baixa? Fatores que o destinatário vê antes de abrir influenciam: Nome do Remetente, Endereço de E-mail, Reconhecimento da Marca, Linha de Assunto e Prévia do Texto. Trabalhe para gerar confiança e clareza nesses elementos.

Uma taxa de reclamação (spam) de 0,05% é aceitável? Embora alguns provedores permitam até 0,05%, é melhor manter essa taxa o mais baixa possível, idealmente abaixo de 0,03%. Taxas próximas a 0,04% já podem começar a prejudicar a reputação do seu domínio.

Como posso melhorar minhas taxas de clique e resposta? * Para Cliques: Tenha uma Chamada para Ação (CTA) clara, única e de valor. Diga exatamente o que o destinatário ganha ao clicar. * Para Respostas: Faça perguntas simples (ex: sim/não) ou ofereça algo valioso em troca da resposta. Mantenha o foco do e-mail no pedido de resposta, evitando links concorrentes.

Com que frequência devo verificar essas estatísticas? Recomenda-se uma verificação semanal. Isso permite que você identifique tendências, corrija problemas rapidamente e otimize continuamente o desempenho das suas campanhas.

O objetivo é atingir os números "Bons" de qualquer jeito? Não exatamente. Os números fornecidos são diretrizes. O objetivo principal é melhorar em relação ao seu ponto de partida. Faça um pequeno ajuste de cada vez (ex: testar uma nova linha de assunto), avalie o impacto e continue iterando para melhorar seus resultados de forma sustentável.

Blocos RSS no Construtor de E-mails

Como fazer

Visão Geral Os elementos RSS no Construtor de E-mails do Full Funnel permitem que você envie automaticamente e-mails com conteúdo inserido dinamicamente sempre que um novo item de um feed RSS (Real Simple Syndication) for publicado.

Usando o Bloco de Cabeçalho RSS Este bloco preenche dinamicamente as tags <channel> do seu feed RSS. Ele suporta as seguintes tags de canal:

Tag RSS Valor Personalizado no Full Funnel
<title> {{rss_feed.title}}
<description> {{rss_feed.description}}
<link> {{rss_feed.url}}
<lastBuildDate> {{rss_feed.date}}

Ao adicionar o bloco, você verá duas opções no seletor "Opções de Edição RSS": * Básico: Insere um texto não editável no editor. * Personalizado: Insere um texto editável no editor que você pode modificar.

Usando o Bloco de Itens RSS Este bloco insere os itens individuais do seu feed. Ele aceita os seguintes valores personalizados:

Tag RSS / Informação Valor Personalizado Descrição
<title> {{rss_item.title}} O título do item (ex.: post do blog).
<description> {{rss_item.content}} Um resumo do item em HTML. Usa <description> ou <content:encoded>.
<link> {{rss_item.url}} O link para o item online.
<content:encoded> {{rss_item.content_full}} O conteúdo completo do item em HTML. Usa <content:encoded> ou <description>.
pubDate {{rss_item.date}} Data de publicação no formato MMM DD, AAAA hh:mm A.
<dc:creator> {{rss_item.author}} O autor do item.
<media:content> {{rss_item.imageUrl}} A URL da imagem. Para renderizá-la, use {{rss_img}}.
Abertura do item {{#rss_items rss_items}} Abre a formatação para os itens. Não exibe conteúdo.
Fechamento do item {{/rss_items}} Fecha a formatação para os itens. Não exibe conteúdo.

Assim como no cabeçalho, o bloco de itens oferece opções Básica (texto pré-formatado não editável) e Personalizada (texto editável).

Feed RSS Baseado em HTML Os valores retornados por variáveis como {{rss_item.title}} são escapados em HTML (ex.: & vira &amp;). Se seu feed contém HTML e você quer que ele seja renderizado, use três chaves {{{ em vez de duas. Exemplo: {{{rss_item.content}}}.

Agendando um E-mail com RSS 1. No Construtor de E-mails, vá para a aba "Enviar ou Agendar". 2. Selecione a opção "Campanha de E-mail RSS". 3. Dê um nome à campanha no campo "Nome da Campanha". 4. Cole a URL do seu feed RSS no campo "URL do Feed RSS". 5. Em "Quando devemos enviar", defina a frequência (Diariamente, Semanalmente, Mensalmente) e o horário. 6. Em "Enviar em", escolha os dias da semana (para frequência semanal). 7. Defina os destinatários e revise as informações no painel lateral. 8. Confirme o agendamento.

Testando E-mails RSS Ao usar a função "Enviar E-mail de Teste" em um modelo com elementos RSS, um novo campo "URL do Feed RSS" aparecerá. Você deve fornecer uma URL de feed de teste aqui, caso contrário, todas as variáveis RSS serão substituídas por texto vazio.

Visualizando e Editando Campanhas RSS As campanhas RSS agendadas aparecem na aba "Agendados", com o tipo listado como "RSS". Lá você pode: * Editar: Clica no ícone de lápis. Uma confirmação perguntará se você deseja cancelar o agendamento atual e criar um novo. Ao confirmar, o agendamento atual é cancelado e o Construtor de E-mails abre com as opções de RSS pré-preenchidas. * Pausar/Retomar: Use as ações de pausa e retomada. * Excluir: Remove permanentemente o agendamento.

Requisitos

  • Um feed RSS válido e acessível publicamente.
  • Um modelo de e-mail criado no Construtor de E-mails do Full Funnel.
  • Uma lista de contatos ou segmento para enviar os e-mails.

FAQ

Os e-mails RSS podem ser agendados como uma campanha normal (enviar agora, gotejamento)? Não. Se você tentar agendar um modelo com elementos RSS usando as opções normais (enviar agora, agendar para depois, modo gotejamento), todas as variáveis personalizadas baseadas em tags RSS serão substituídas por texto vazio. Você deve usar a opção específica "Campanha de E-mail RSS" na aba "Enviar ou Agendar".

Onde vejo minhas campanhas RSS agendadas? Na aba "Agendados" da sua conta. O tipo da campanha será listado como "RSS", diferente das campanhas normais.

Como faço para que o HTML do meu feed RSS seja renderizado no e-mail? Use três chaves {{{ em vez de duas ao chamar a variável no construtor. Por exemplo: {{{rss_item.content}}}.

Posso usar as variáveis RSS no assunto do e-mail? Sim. Você pode usar as quatro variáveis do cabeçalho ({{rss_feed.title}}, {{rss_feed.description}}, etc.) no campo de assunto do e-mail.

Importando Contatos Duplicados com Mesmo E-mail e Telefone

Como fazer

O sistema do Full Funnel possui uma lógica específica para evitar a criação de contatos duplicados durante uma importação via CSV. Ao importar um arquivo, o sistema verifica os registros existentes com base no endereço de e-mail e no número de telefone. Se um novo registro a ser importado tiver um e-mail ou telefone que já exista na sua base de contatos, as informações desse novo registro serão mescladas ao contato existente, em vez de criar um duplicado.

Este processo de mesclagem é automático e garante que suas listas de contatos permaneçam organizadas e sem entradas redundantes.

Requisitos

  • Acesso de Administrador: É necessário ter permissões de administrador na sua conta do Full Funnel para realizar importações.
  • Arquivo CSV Válido: O arquivo a ser importado deve estar no formato CSV (Valores Separados por Vírgula) e estruturado corretamente, com colunas de cabeçalho claras.
  • Contatos Existentes: A lógica de mesclagem só se aplica se já houver um ou mais contatos na sua base com o mesmo e-mail ou telefone presente no arquivo de importação.

FAQ

O que acontece com os dados do novo registro durante a mesclagem? Os dados do novo registro importado sobrescrevem as informações do contato existente nos campos correspondentes. Campos vazios no novo registro não apagarão dados existentes. Campos com informações novas (que o contato antigo não possuía) serão adicionados ao perfil.

É possível forçar a criação de um contato duplicado? Não, a lógica de prevenção de duplicatas com base em e-mail e telefone é automática e não pode ser desativada para importações padrão. A recomendação é revisar e limpar seu arquivo CSV antes da importação.

Como importar contatos sem que eles sejam mesclados? Para que um registro seja criado como um novo contato, ele não pode compartilhar o mesmo endereço de e-mail ou o mesmo número de telefone com nenhum contato já existente na sua base. Certifique-se de que os dados de e-mail e telefone no seu CSV sejam únicos.

Elemento de Vídeo no Construtor de E-mails do Full Funnel

O que é o Elemento de Vídeo?

O Elemento de Vídeo no Construtor de E-mails do Full Funnel permite adicionar uma prévia de vídeo clicável aos seus e-mails. Como a reprodução direta de vídeo não é suportada pela maioria dos clientes de e-mail, este elemento cria uma miniatura com um botão de "play". Quando o contato clica na imagem, ele é direcionado para a URL do vídeo (no YouTube, Vimeo, etc.).

Importante: Por limitações técnicas dos provedores de e-mail, não é possível incorporar vídeos que reproduzam diretamente dentro do e-mail. Este bloco cria um link visual para o seu vídeo.

Como fazer

Para localizar e usar o Elemento de Vídeo:

  1. Acesse sua subconta do Full Funnel.
  2. No menu lateral, vá até Marketing > E-mails.
  3. Clique em + Criar Campanha, escolha a opção Em Branco e depois Editor de Design.
    • Alternativamente, você pode abrir um Modelo de E-mail existente para edição.
  4. No painel esquerdo, localize o Elemento de Vídeo na seção Elementos.
  5. Arraste e solte o elemento de Vídeo no layout do seu e-mail.

Configurações do Editor de Vídeo

Ao clicar no elemento de vídeo adicionado, as seguintes configurações estarão disponíveis no painel direito:

  • Tipo de Vídeo: Selecione a origem do seu vídeo. As opções são: YouTube, Vimeo, Wistia e HTML5.
    • O sistema tentará buscar a miniatura padrão do vídeo com base na URL fornecida.
    • O estilo do botão de "play" pode mudar conforme o provedor (ex.: fundo vermelho para YouTube).
  • URL do Vídeo: Insira aqui o link completo do seu vídeo (ex.: https://youtu.be/seuID).
    • Se o tipo de vídeo selecionado não corresponder à URL, um aviso será exibido.
  • URL da Miniatura do Vídeo:
    • Este campo é preenchido automaticamente com a miniatura buscada da URL do vídeo.
    • Você pode usar uma miniatura personalizada:
      • Clique em Substituir para fazer upload de uma nova imagem.
      • Clique em Remover para apagar a miniatura atual.
  • Largura & Altura: Defina as dimensões do elemento de vídeo no e-mail.
    • Se deixado em branco, o vídeo ocupará a largura total do container.
    • Nota: Ao definir dimensões personalizadas, o elemento será alinhado à esquerda. Para centralizá-lo, use a configuração de Preenchimento (Padding) ou coloque o vídeo dentro de um layout de colunas.
  • Preenchimento (Padding): Adicione espaçamento interno ao redor do elemento de vídeo para melhor posicionamento no layout.

Requisitos

  • O Elemento de Vídeo está disponível apenas dentro do Editor de Design do Construtor de E-mails e nos Modelos de E-mail.
  • É necessário ter a URL pública de um vídeo hospedado em uma das plataformas suportadas (YouTube, Vimeo, Wistia) ou um link direto para um arquivo de vídeo (HTML5).

FAQ

P: O vídeo vai tocar automaticamente dentro do e-mail? R: Não. Os clientes de e-mail não suportam reprodução de vídeo incorporada. O elemento cria uma imagem de prévia clicável que leva o contato para a página onde o vídeo está hospedado.

P: Posso usar uma miniatura personalizada? R: Sim. Após adicionar a URL do vídeo, você pode clicar em "Substituir" no campo URL da Miniatura do Vídeo para fazer upload da sua própria imagem.

P: Como centralizar o vídeo no e-mail? R: Se o vídeo com dimensões personalizadas estiver alinhado à esquerda, você pode: 1. Ajustar o Preenchimento (Padding) para movê-lo. 2. Arrastar o elemento de vídeo para dentro de um bloco de Colunas (2 ou 3 colunas) no editor, que oferece mais opções de alinhamento.
