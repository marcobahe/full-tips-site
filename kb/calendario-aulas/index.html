Calendários de Classe

Calendários de Classe são projetados para gerenciar eventos de um-para-muitos, onde vários convidados podem participar de uma única reserva. Ideais para webinars, workshops, sessões de treinamento e eventos esportivos. Em um Calendário de Classe, só pode haver um proprietário da consulta, normalmente o instrutor ou anfitrião. O número de vagas por horário pode ser definido, com um limite máximo de 9.999 participantes, embora existam limitações quanto ao local da reunião.

O que é uma Reserva de Classe?

Uma reserva de classe é um tipo de calendário de um-para-muitos para hospedar vários convidados em uma única reserva. As reservas de classe são úteis quando você deseja que vários convidados participem de um único evento ao mesmo tempo; alguns exemplos:

  • Webinars/Workshops
  • Sessões de Treinamento
  • Eventos Esportivos

Por que usar Calendários de Reserva de Classe?

O uso de Calendários de Classe oferece vários benefícios para gerenciar eventos com vários participantes:

  • Organização eficiente: Calendários de Classe simplificam a organização e o gerenciamento de eventos de um-para-muitos, permitindo que você crie uma única reserva que acomoda vários convidados, agilizando o processo de agendamento.
  • Capacidade personalizável: Você pode definir o número de vagas por horário para controlar o tamanho da classe e garantir um ambiente ideal de aprendizado ou participação.
  • Gerenciamento centralizado: Calendários de Classe fornecem uma plataforma centralizada para gerenciar reservas, cancelamentos e reagendamentos, tornando mais fácil para o proprietário da consulta supervisionar todos os aspectos do evento.
  • Comunicação aprimorada: Calendários de Classe facilitam uma melhor comunicação com os participantes, fornecendo uma visão geral clara dos detalhes do evento, como data, hora e local. Isso ajuda a reduzir a confusão e a probabilidade de não comparecimentos.
  • Economia de tempo: Ao automatizar o processo de reserva e fornecer opções para os participantes cancelarem ou reagendarem suas consultas, os Calendários de Classe economizam tempo tanto para o organizador quanto para os participantes.

Como enviar um e-mail para todos os participantes registrados de uma classe

  1. Vá para Contatos.
  2. Filtre seus contatos pela consulta de classe específica (usando Filtros Avançados ou filtros baseados em Consulta).
  3. Selecione os contatos na lista de participantes exibida.
  4. Use a ação em massa para enviar um e-mail.

Você também pode criar uma Lista Inteligente filtrada por essa consulta de classe se planeja se comunicar com esse grupo mais de uma vez.

Como cancelar ou reagendar uma consulta de classe para um único participante

  1. Vá para Calendários.
  2. Localize a consulta de classe desejada.
  3. Clique no menu de opções (três pontos) ao lado do contato.
  4. Selecione a opção para cancelar ou reagendar a consulta para contatos específicos.

Como reservar uma consulta de classe diretamente com contatos

  1. Vá para Calendários.
  2. Clique no botão "Agendar Consulta".
  3. Selecione o contato com o qual deseja agendar a consulta.
  4. Selecione o calendário (escolha o calendário de reserva de classe aqui).
  5. Selecione o horário.
  6. Prossiga para reservar a consulta.

Como permitir que os participantes cancelem/reagendem uma consulta de Reserva de Classe

  1. Vá para Configurações > Calendários.
  2. Edite o calendário de classe desejado.
  3. Vá para a aba Confirmação.
  4. Habilite a opção "Permitir Cancelamento" e "Permitir Reagendamento".

Como cancelar uma sessão de classe inteira com todos os participantes

  1. Vá para Calendários.
  2. Abra a consulta de classe na Visualização de Calendário ou na Visualização de Lista de Consultas.
  3. Clique na opção Excluir (ícone de lixeira).
  4. Confirme a exclusão quando solicitado.

Esta ação remove a sessão de classe e todos os participantes associados desse horário.

Requisitos

  • É importante notar que, ao compartilhar o link do Calendário de Reserva de Classe, ele sempre abre no mês atual, não no mês do seu evento. Se sua classe estiver agendada para um mês diferente, o calendário pode mostrar "Nenhum horário disponível" por padrão. Os usuários precisarão clicar na seta para navegar até o mês do evento.

Perguntas Frequentes

P: Posso enviar um e-mail avulso para todos os participantes registrados de uma classe? R: Sim, vá para Contatos e filtre seus contatos pela consulta de classe específica (usando Filtros Avançados ou filtros baseados em Consulta). Depois que a lista de participantes registrados for exibida, selecione os contatos e use a ação em massa para enviar um e-mail. Você também pode criar uma Lista Inteligente filtrada por essa consulta de classe se planeja se comunicar com esse grupo mais de uma vez.

P: Posso cancelar ou reagendar uma consulta de classe para um único participante? R: Sim, para consultas de classe, você pode clicar no menu de opções ao lado do contato, onde você pode ver a opção de cancelar ou reagendar a consulta para contatos específicos.

P: Posso reservar uma consulta de classe diretamente com contatos? R: Sim, você pode fazer isso; clique no botão "Agendar Consulta" e selecione o contato com quem deseja agendar a consulta. Em seguida, selecione o calendário (escolha o calendário de reserva de classe aqui), selecione o horário e, em seguida, prossiga para reservar a consulta.

P: Os participantes podem cancelar/reagendar uma consulta de Reserva de Classe? R: Para permitir que os participantes cancelem/reagendem a consulta de reserva de classe, habilite a opção "Permitir Cancelamento" e "Permitir Reagendamento" na aba Confirmação em Configurações do Calendário.

P: Como cancelo uma sessão de classe inteira com todos os participantes? R: Para cancelar a sessão de classe inteira (incluindo todos os participantes registrados), vá para Calendários, abra a consulta de classe na Visualização de Calendário ou na Visualização de Lista de Consultas e clique na opção Excluir (ícone de lixeira). Confirme a exclusão quando solicitado. Esta ação remove a sessão de classe e todos os participantes associados desse horário.
