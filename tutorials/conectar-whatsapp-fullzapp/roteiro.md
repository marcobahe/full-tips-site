# Roteiro: Como Conectar e Configurar o Fullzapp

## 📋 Informações Gerais
- **Título:** Como Conectar seu WhatsApp e Configurar o Fullzapp
- **Público:** Usuário leigo, totalmente iniciante
- **Tempo estimado:** ~4 min
- **Passos:** 12

---

## Passo a Passo

### Passo 1 — Acesse o menu "Fullzapp"
- **Onde:** Menu lateral esquerdo → ícone/menu "Fullzapp"
- **Screenshot:** Página inicial do Full Funnel com sidebar expandido. Destacar o item "Fullzapp" no menu lateral.
- **Hotspot:** Sobre o item "Fullzapp" no menu
- **Tooltip:** "No menu lateral esquerdo, clique em **Fullzapp**. É por aqui que você conecta seu WhatsApp à plataforma. O Fullzapp é o integrador que permite enviar e receber mensagens do WhatsApp diretamente pelo Full Funnel."
- **Dica:** "💡 O Fullzapp funciona com qualquer número de WhatsApp — pessoal ou Business. Você não precisa ter a API Oficial da Meta."

### Passo 2 — Visualize suas conexões
- **Onde:** Tela principal do Fullzapp (listagem de conexões/números)
- **Screenshot:** Página do Fullzapp mostrando a lista de números/conexões. Se já tem algum conectado, aparece aqui.
- **Hotspot:** Sobre o botão "Adicionar número" ou "Conectar novo número" (canto superior direito ou área principal)
- **Tooltip:** "Esta é a tela de gerenciamento das suas conexões WhatsApp. Aqui você vê todos os números conectados, o status de cada um e pode adicionar novos. Para conectar um número, clique no botão **'Adicionar'** ou **'Conectar novo número'**."
- **Dica:** "💡 Você pode conectar múltiplos números de WhatsApp na mesma conta. Cada número funciona de forma independente."

### Passo 3 — Clique para adicionar novo número
- **Onde:** Botão de adicionar/conectar número
- **Screenshot:** Mesmo da tela anterior, com destaque no botão de ação
- **Hotspot:** No botão "Adicionar" / "+" / "Conectar"
- **Tooltip:** "Clique no botão para iniciar a conexão de um novo número. Na próxima tela, o sistema vai gerar um QR Code para você escanear com seu celular."
- **Dica:** null

### Passo 4 — QR Code aparece na tela
- **Onde:** Modal ou tela com o QR Code gerado
- **Screenshot:** QR Code visível na tela, grande e claro
- **Hotspot:** Sobre o QR Code
- **Tooltip:** "O QR Code foi gerado! Agora você precisa escaneá-lo com o WhatsApp do seu celular. **Não feche esta tela** — o QR Code expira em alguns minutos. Se expirar, basta clicar em 'Gerar novamente'."
- **Dica:** "⚠️ Importante: o número que você vai conectar **não pode estar aberto no WhatsApp Web** em outro computador. Se estiver, desconecte primeiro (WhatsApp > Dispositivos conectados > Sair)."

### Passo 5 — Abra o WhatsApp no celular
- **Onde:** Instrução visual (pode ser um mockup ou texto instrucional sobre o screenshot)
- **Screenshot:** Tela do QR Code com instruções ao lado ou overlay explicativo
- **Hotspot:** Área das instruções
- **Tooltip:** "No seu celular, abra o **WhatsApp** e siga estes passos:\n\n**Android:** Toque nos 3 pontinhos (⋮) no canto superior direito → **Dispositivos conectados** → **Conectar dispositivo**\n\n**iPhone:** Vá em **Configurações** (engrenagem) → **Dispositivos conectados** → **Conectar dispositivo**"
- **Dica:** "💡 Se o seu celular pedir autenticação (digital, Face ID ou PIN), é normal — é uma camada de segurança do próprio WhatsApp."

### Passo 6 — Escaneie o QR Code
- **Onde:** Mesma tela do QR Code
- **Screenshot:** QR Code na tela com indicação visual de "aponte a câmera aqui"
- **Hotspot:** Sobre o QR Code
- **Tooltip:** "Aponte a câmera do celular para o QR Code na tela do computador. O WhatsApp vai reconhecer automaticamente. **Mantenha o celular firme** por 2-3 segundos até o código ser lido."
- **Dica:** null

### Passo 7 — Conexão estabelecida!
- **Onde:** Tela de confirmação / status "Conectado"
- **Screenshot:** Tela do Fullzapp mostrando o número com status verde/conectado
- **Hotspot:** Sobre o indicador de status "Conectado" (bolinha verde ou texto)
- **Tooltip:** "Pronto! Seu WhatsApp está conectado ao Full Funnel! 🎉 O status deve aparecer como **'Conectado'** (verde). A partir de agora, todas as mensagens enviadas e recebidas por esse número aparecem na seção de **Conversas** da plataforma."
- **Dica:** "💡 O celular precisa ficar com internet ativa. Se o celular ficar sem conexão por muito tempo, o WhatsApp pode desconectar automaticamente."

### Passo 8 — Conexão pronta! Agora vamos configurar os recursos extras
- **Onde:** Tela do Fullzapp com número conectado, mostrando os botões de configuração
- **Screenshot:** Tela do Fullzapp com status "Conectado" e os botões/toggles de configuração visíveis
- **Hotspot:** Área geral dos botões de configuração
- **Tooltip:** "Seu WhatsApp está conectado! 🎉 Agora vamos ativar os recursos extras do Fullzapp. São 4 configurações opcionais que vão turbinar seu uso. Cada uma é um botão que você liga ou desliga. Vamos ver uma por uma."
- **Dica:** null

### Passo 9 — Ativar alerta de desconexão
- **Onde:** Botão/toggle de "Notificar quando desconectar" (ou nome similar)
- **Screenshot:** Destaque no botão de alerta de desconexão
- **Hotspot:** Sobre o botão/toggle de notificação de desconexão
- **Tooltip:** "Ative esta opção para ser **avisado automaticamente** caso seu número desconecte do Fullzapp. Isso pode acontecer se o celular ficar sem internet por muito tempo, se o WhatsApp for atualizado, ou se alguém desconectar manualmente. Com o alerta ativo, você fica sabendo na hora e pode reconectar rapidamente."
- **Dica:** "💡 Recomendamos fortemente deixar esta opção **sempre ativada**. Se o número desconectar e você não souber, suas automações e mensagens vão parar de funcionar sem aviso."

### Passo 10 — Ativar identificação do usuário nas mensagens
- **Onde:** Botão/toggle de "Identificação do remetente" (ou nome similar)
- **Screenshot:** Destaque no botão de identificação de usuário
- **Hotspot:** Sobre o botão/toggle de identificação
- **Tooltip:** "Quando você tem uma equipe com vários atendentes usando a plataforma, esta opção **identifica qual usuário enviou cada mensagem**. O destinatário no WhatsApp vê o nome do atendente junto com a mensagem, assim sabe com quem está falando. Muito útil para equipes de suporte e vendas."
- **Dica:** "💡 Se você é o único usuário da conta, pode deixar desativado. Mas se tem equipe, ative — seus clientes vão saber exatamente com quem estão conversando."

### Passo 11 — Ativar transcrição automática de áudios
- **Onde:** Botão/toggle de "Transcrição de áudio" + campo para API Key da OpenAI
- **Screenshot:** Destaque no botão de transcrição e no campo de API Key
- **Hotspot:** Sobre o botão/toggle de transcrição de áudios
- **Tooltip:** "Este recurso transcreve automaticamente os áudios que seus contatos enviam pelo WhatsApp! Em vez de ouvir cada áudio, você lê o texto transcrito direto na plataforma. Para funcionar, você precisa inserir uma **chave de API da OpenAI** (a transcrição usa o Whisper da OpenAI). Cole sua API Key no campo indicado e ative."
- **Dica:** "💡 A API da OpenAI tem um custo por minuto de áudio transcrito, mas é bem acessível (centavos por áudio curto). Se você recebe muitos áudios de clientes, este recurso economiza MUITO tempo — chega de ouvir áudio de 3 minutos pra descobrir que era só um 'ok'! 😄"

### Passo 12 — Ativar notificações de grupos do WhatsApp
- **Onde:** Botão/toggle de "Notificações de grupo" (entrada/saída de membros)
- **Screenshot:** Destaque no botão de notificações de grupo
- **Hotspot:** Sobre o botão/toggle de notificações de grupo
- **Tooltip:** "Se você administra grupos no WhatsApp, esta opção avisa quando **pessoas entram ou saem** dos seus grupos. Funciona apenas para grupos onde seu número conectado é administrador. Útil para acompanhar a movimentação dos seus grupos sem precisar abrir o WhatsApp no celular o tempo todo."
- **Dica:** "💡 Ideal para quem usa grupos como comunidade de clientes, alunos ou membros. Você monitora tudo direto pela plataforma."

- **Blur obrigatório:** Números de telefone, nomes de contatos, fotos de perfil, API Keys
- **Sidebar expandido** em todas as telas (regra do Marco)
- **Interface em PT-BR**
- Passos 4, 5 e 6 podem reutilizar o mesmo screenshot com hotspots diferentes
- Passos 8-12 provavelmente reutilizam a mesma tela com hotspots nos 4 botões
- O passo 5 é instrucional (celular) — pode ser overlay/texto sobre a tela do QR

- Não assumir nada — pegar na mão do início ao fim
